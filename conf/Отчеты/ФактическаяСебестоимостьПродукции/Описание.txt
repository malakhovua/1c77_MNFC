Данный отчет в самых различных разрезах позволяет получить 
информацию о фактической себестоимости выпущенной продукции.

Отчет может быть сформирован:
	- по продукции, выпущенной в определенном подразделе-
нии;
	- по продукции, выпущенной в рамках определенного вида 
деятельности;
	- по продукции, выпущенной по определенному заказу;
	- по продукции определенного вида;
	- по продукции, содержащей в своем составе определенную 
статью калькуляции или материал;
	- по продукции, определяемой произвольным списком, со-
ставленным по одному из перечисленных признаков 

Информация в отчете может быть представлена в разрезе фирм 
компании (если, разумеется, отбор по фирме не был установлен), 
видов деятельности, подразделений, видов продукции, заказов, ста-
тей калькуляции, материалов.
 
С помощью флажков группы "Данные в отчете" можно управлять 
составом граф, включаемых в его форму. Если все флажки включе-
ны, информация в отчете представлена в следующем виде:
	Нач. остаток -стоимость незавершенного производства на нача-
ло указанного периода. 
	Приход -общая стоимость накопленных за период производст-
венных затрат. 
	Расход -стоимость затрат, включенных в себестоимость продук-
ции. 
	Кон. остаток- стоимость незавершенного производства на конец 
указанного периода. 

С помощью флажков "Себестоимость единицы" и "Выделять да-
вальческую составляющую" для каждой единицы продукции может 
быть представлена как ее полная себестоимость и ее давальческая 
составляющая.

При определении состава группировок, отборов и уровня детали-
зации отчета необходимо представлять какие данные и в какой дета-
лизации вы хотите получить. Например, если нужно определить, ка-
кая сумма каждого вида затрат входит в единицу каждой продукции, 
выпущенной в подразделении "Цех №1", настройка отчета должна 
выглядеть следующим образом. В реквизите "Подразделение" нужно 
выбрать подразделение "Цех №1", а в группе детализация - вклю-
чить позиции "Номенклатура" и "Вид затрат", разместив их после-
довательно друг за другом.

С помощью кнопки "Формы" можно установить предопределен-
ные настройки отчета: 
	- "Анализ по продукции" -устанавливает детализацию по 
единственной позиции "Номенклатура";
	- "Анализ продукции по статьям калькуляции"  - устанав-
ливает детализацию по позициям "Номенклатура" и 
"Статья калькуляции";
	- "Анализ продукции по заказам" - устанавливает детали-
зацию по позициям "Номенклатура" и "Заказ";
	- "Анализ по подразделениям" - устанавливает детализа-
цию по позиции "Подразделение";
	- "Анализ подразделений по статьям калькуляции" - уста-
навливает детализацию по позициям "Подразделение" и 
"Статья калькуляции";
	- "Анализ по статьям калькуляции" - устанавливает дета-
лизацию по позициям "Подразделение" и "Статья каль-
куляции";
	- "Анализ по видам деятельности"  - устанавливает дета-
лизацию по позиции "Вид деятельности".
