	Это универсальный (т. е. работающий в любой конфигурации) отчет,
который позволяет выводить на печать результат запроса по регистру,
заданному в конфигурации справочника. 
	Сначала Вы выбираете регистр. Затем выбираете период, за который 
будет формироваться отчет.  При выборе режима "до ТА"  поле ввода 
конечной даты недоступно, а данные выводятся от начальной даты до 
даты точки актуальности.
	Отчет выполняется в двух вариантах-
вывод итогов по регистрам и вывод движений регистров.
	Для построения отчета по итогам необходимо перейти на закладку "Итоги".
Далее, при помощи соответствующих элементов управления, определяете, 
как должны группироваться данные, и состав ресурсов регистра, по которым 
будут вычисляться функции. Отмечая нужные функции флажками, можете 
изменить состав агрегатных функций,которые будут вычисляться (по умолчанию 
выбраны все доступные функции:остатки на начало и на конец периода, приход
и расход за период).
	Для получения отчета по движениям необходимо перейти на закладку 
"Движения". В списке измерений, ресурсов и реквизитов выбрать те, которые 
выводятся в отчет. Помечать весь список можно по кнопке "Вкл. Все" и 
отключить все по кнопке "Откл. Все".
	Для каждого вида отчета можно задать условия отбора значений по 
всем измерениям и ресурсам выбранного регистра. Для задания условий 
необходимо перейти на закладку "Условия". По кнопке "Добавить" - появляется 
диалог для выбора условий. В диалоге необходимо выбрать нужный атрибут 
в поле "Данные", проставить условие и в поле "Значение" определить нужное 
значение отбора. Если условие должно использоваться при формировании 
отчета, проставить признак "Использовать условие". Для  добавлении
условия в таблицу формы отчета, нажать на кнопку "ОК", для отмены выбора - 
кнопка "Отмена".Добавленные условия можно включать или исключать из 
условий формирования отчета, изменяя колонку "+"("+" - включить в запрос). Чтобы 
отредактировать условие в таблице, необходимо щелкнуть два раза по нужной 
колонке и появившемся диалоге условий изменить параметры и нажать на кнопку "ОК".
	Разные типы отчетов формируются нажатием на кнопку "Сформировать" на 
соответствующих закладках "Итоги" или "Движения".
	Полученая табличная часть может быть использована,
например, для дальнейшего анализа данных в таблицах Excel (для этого
ее надо сохранить в формате MS Excel).  
	Настройки элементов формы отчета можно сохранять, формируя список настроек.
Для сохранения настроек необходимо перейти на закладку "Настройки". Если нет ни 
одной сохраненной настройки, в поле "Настройки" появляется наименование по умолчанию 
- "Основной". Это наименование можно изменить и нажать на кнопку "Сохранить". 
Тогда в список настроек добавится наименование сохраненной настройки. 
Для восстановления элементов нужной настройки, необходимо выбрать
эту настройку из списка и нажать на кнопку "Восстановить". Для удаления сохраненной 
настройки необходимо выбрать эту настройку из списка и нажать на кнопку "Удалить".
