Для сравнения нормативной и фактической себестоимости раз-
личных видов продукции, выпущенных в течение месяца, предназна-
чен отчет "Плановая и фактическая себестоимость продукции". 

Отчет может быть сформирован за период времени произвольной 
длительности, дискретностью которого является месяц.
 
С помощью единичного и множественного фильтров в него мож-
но отобрать информацию в самых различных разрезах: по всем ви-
дам выпущенной продукции по конкретному виду деятельности или 
подразделению, для конкретного вида продукции, по конкретным 
заказам, статьям калькуляции, материалам.

Видимо, наиболее используемым вариантом детализации отчета 
является детализация отчета по наименованиям продукции и стать-
ям калькуляции. Именно он предлагается по умолчанию при первом 
открытии формы отчета.

Сравнительная информация может быть представлена в отчете в 
двух вариантах:
	- для всего объема каждого вида выпущенной продукции;
	- для единицы продукции каждого вида.

Дополнительно в отчет можно также включить графу, в которой 
сумма отклонения по текущей строке будет представлена в суммо-
вом и процентом выражении. 

Для управления составом включаемых в отчет данных служат 
флажки группы "Данные в отчете". 

Для материальных затрат (материалов) параллельно с себестои-
мостью можно выполнить сопоставление их нормативного и факти-
ческого количества, списанного на производство того или иного вида 
продукции. Для этого в группе "Данные в отчете" нужно включить 
флажок "Кол-во материалов".
