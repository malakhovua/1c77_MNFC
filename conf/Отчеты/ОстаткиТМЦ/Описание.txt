Данный отчет предназначен для отображения информации об ос-
татках или движениях ТМЦ на складах в количественном и суммо-
вом выражении по себестоимости.

В зависимости от выбранной настройки вида отчета, можно каж-
дый из них представить в виде трех различных отчетов: остатки 
ТМЦ, движение ТМЦ, стоимость ТМЦ на складах. Для последнего 
вида отчета дополнительно устанавливается тип цен ТМЦ, которые 
будут представлены для каждого ТМЦ в отдельной графе отчета.

Отчет "Остатки ТМЦ" выводит полный или частичный (если в 
диалоге выбран только одна номенклатурная позиция, группа или 
категория ТМЦ) список ТМЦ и их остатки на всех или только на 
выбранных складах. Расчет остатков производится на дату оконча-
ния отчета. Существует возможность произвести дополнительный 
отбор по количественным характеристикам ТМЦ (только для отче-
тов по остаткам ТМЦ):

Показать все ненулевые остатки - будут отобраны ТМЦ, ко-
личество которых не равно нулю, включая и отрица-
тельные остатки на конкретном складе.

Показать все - будут отобраны все ТМЦ в соответствии с уста-
новленным фильтром.

Показать все, имеющиеся в наличии - будут отобраны только 
те ТМЦ, суммарное количество которых на выбран-
ных больше нуля.

Показать меньше мин. остатка - в отчет будут включены 
только те ТМЦ, количество которых меньше мини-
мального товарного остатка. Расчет количества про-
изводится так же, как и в предыдущем случае.

Показать меньше мин. остатка с учетом резерва - в отчет 
будут включены только те ТМЦ, количество которых 
меньше, чем минимальный остаток минус зарезерви-
рованное количество товара. Расчет количества про-
изводится так же, как и в предыдущем случае.

С помощью фильтров в отчет может быть включена информация:
	- по ТМЦ на конкретном складе (или на нескольких скла-
дах с помощью множественного фильтра);
	- по ТМЦ отдельного вида ТМЦ (или видов ТМЦ);
	- по отдельным категориям ТМЦ.

Информация, выводимая в отчете:
	Товар - полное наименование ТМЦ с разбиением на группы, 
здесь же приведена информация о неприкосновенном 
запасе на складе, указанная для товара в справочнике 
(Мин. остаток). 
	Резерв - общее количество зарезервированного товара в 
управленческом отчете. Резерв считается по 
компании в целом.
	Название склада / Остаток - реальное количество товара на 
складе.
