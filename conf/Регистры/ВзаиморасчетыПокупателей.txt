Регистр "Взаиморасчеты покупателей" предназначен для 
ведения учета взаиморасчетов с покупателями. Данный регистр 
используется при формировании отчетов по взаиморасчетам с 
контрагентами.

Структура регистра.

Измерения:
	Фирма - Здесь проставляется значение фирмы
	Контрагент - Здесь проставляется значение контрагента.
	Договор - Здесь проставляется значение договора.
	СтавкаНДС - Здесь проставляется значение ставки НДС 
текущего взаиморасчета.
	Счет - Документ, по которому проводятся взаиморасчеты - 
Счет или накладная, в которой ДокументОснование не указан.
	КредДокумент - Здесь проставляется значение документа, 
который образовал кредит по взаиморасчетам.
	Валюта - Валюта долга, обычно совпадает с валютой 
документа, сформировавшего движение по регистру.

Ресурсы:
	Долг - На этом ресурсе ведется учет взаиморасчетов с покупателем по данному 
кредиту. В валюте долга (см. измерение Валюта). Знак этого ресурса, если контрагент 
должен фирме - положительный (долг контрагента), иначе знак отрицательный.
	Долг осн. - Долг в базовой валюте (гривнях).

Реквизиты:
	Код операции. Односимвольное строковое поле, в котором проставляется символьный 
код выполненной операции.             
	Флаг НУ - Односимвольное строковое поле, в котором проставляется символьный код 
операции по налоговому учету. 
	Сумма с НДС НУ - Сумма с НДС, которая отражается в налоговом учете.
	НДС - Сумма НДС по текущей операции.
