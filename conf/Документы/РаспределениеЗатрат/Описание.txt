Для выполнения операций распределение общепроизводственных 
затрат и затрат на брак, предназначен документ "Распределение за-
трат". С помощью этого документа выполняется также произвольное 
распределение затрат между объектами аналитики в случаях, когда в 
качестве баз распределения затрат нужно использовать не нормы 
затрат, а какие-либо другие производственно-технологические пока-
затели.

Порядок работы с этим документов строго регламентирован. А именно, 
вначале с его помощью следует выполнить распределение 
общепроизводственных затрат и затрат на брак, и только после этого 
приступать к дальнейшему перераспределению затрат между объектами. 

Порядок работы с документом на каждом этапе операции распре-
деления затрат проиллюстрирован в разделе "Порядок распределе-
ния затрат".

Заполнение формы документа следует начать с выбора типа рас-
пределяемых затрат и типа объектов, между которыми их следует 
распределить. 

В реквизите "Тип распределяемых затрат" для выбора предлага-
ется следующий список видов затрат
	- общепроизводственные затраты - затраты, которые на те-
кущий момент отражены только на счете 91 "Общепроиз-
водственные затраты". 
	- затраты на брак - затраты, которые на текущий момент 
отражены на счете 24 "Брак в производстве";
	- материальные затраты - под этим типом затрат понима-
ются уже зарегистрированные в бухгалтерском и опера-
тивном учете затраты по списанию в производство кон-
кретных видов сырья и материалов, которые, например, 
нужно перераспределить между конкретными видами 
продукции и заказами;
	- нематериальные затраты. Под этим типом затрат понима-
ются также прямые производственные затраты, уже заре-
гистрированные в бухгалтерском и оперативном учете, но 
требующие перераспределения между подразделениями, 
видами продукции и пр.

Тип объектов аналитического учета, между которыми нужно рас-
пределить затраты выбирается в реквизите "Тип единицы распреде-
ления". А именно, здесь нужно установить будут ли распределяться 
затраты между: подразделениями, видами деятельности, заказами  
или видами продукциями. Конкретные объекты, между которыми 
должны быть распределены затраты, перечисляются в табличной 
части документа в графе "Единица распределения", которая стано-
вится доступной после выбора типа единиц распределения.

При распределении затрат между объектами учета внутри под-
разделения в форме документа появляется реквизит "Уровень пере-
дела", в котором нужно установить уровень передела продукции, для 
которой будут распределяться затраты. Это значит, что при распре-
делении затрат между видами деятельности, заказами затраты будут 
распределяться только между теми объектами, по которым выпуска-
лась продукция с указанным уровнем передела и ниже. Соответст-
венно при распределении затрат между видами продукции они будут 
распределяться только между видами продукции с указанным уров-
нем передела и ниже.
 
В группе "Отбор затрат" устанавливаются аналитические при-
знаки, по которым должны быть отобраны затраты, подлежащие рас-
пределению (рассчитаны итоги на бухгалтерских счетах (91 или 24) 
или в регистре "Производственные затраты"). Включенная опция 
"по пустому" напротив каждого признака означает, что будут выбра-
ны затраты по "пустому" объекту указанного вида: подразделению; 
виду деятельности, заказу или виду продукции. Такая ситуация со-
ответствует выполнению документом распределения затрат, которые 
были зафиксированы по неопределенным ("пустым") объектам ана-
литики между конкретными объектами. 

В реквизите "Вид затрат", указывается вид затрат, по которому 
отбираются затраты для распределения. Если этот реквизит не за-
полнен, то распределение осуществляется по всем видам затрат.

Если распределяемые затраты должны быть также отнесены на 
другой вид затрат для всех единиц распределения, нужно включить 
опцию "отнести затраты на новый вид затрат", после чего указать 
требуемую статью затрат в реквизите, расположенном справа от это-
го флажка.

Тип базы распределения, пропорционально которой будут рас-
пределяться затраты устанавливается в реквизите "Тип базы". Здесь 
для выбора предлагаются следующие варианты расчета базы:
	- Использовать в качестве базы распределения установлен-
ные для каждой единицы распределения нормы затрат 
(вариант "Нормы производства"). Использовать такой 
тип расчета базы можно например, при распределении за-
трат между видами деятельности внутри подразделения 
пропорционально нормам затрат по каждому виду дея-
тельности.
	- Использовать в качестве базы распределения обороты 
реализации конкретного вида продукции или по конкрет-
ному виду деятельности за определенный период времени, 
который задается в реквизитах "Дата с", "Дата по". В ка-
честве единиц распределения для такого способа расчета 
могут быть выбраны только "Вид деятельности" и "Про-
дукция". Примером использования такого варианта рас-
чета является следующая ситуация. Затраты по виду дея-
тельности "Производство чашек" подразделения "Цех 
№1", которые в течение месяца невозможно было отнести 
на конкретные виды выпускаемой продукции, нужно рас-
пределить между несколькими видами продукции про-
порционально обороту их реализации. В этом случае в ре-
квизите "Тип единицы распределения" должно быть вы-
брано значение "Продукция", в реквизитах "Подразделе-
ние" и "Вид деятельности" группы "Затраты" - "Цех 
№1" и "Производство чашек" соответственно. В таблич-
ной части документа перечисляются наименования про-
дукции, между которыми нужно распределить затраты. 

Информация об оборотах по каждому виду продукции 
выбирается в этом случае из регистра "Обороты".
	- Использовать в качестве базы распределения количество 
выпускаемой продукции. При таком варианте распределе-
ния информация о количестве выпущенной продукции 
выбирается из регистра "Выпуск продукции". Использо-
вать такой тип расчета базы можно например, при распре-
делении затрат между подразделениями пропорционально 
количеству выпущенной продукции в каждом подразде-
лении.
	- Использовать в качестве базы распределения сумму 
фактических затрат по конкретной статье калькуляции, ко-
торая задается в реквизите "Базовая статья калькуляции". 
Такой вариант расчета можно использовать, например, 
для распределения затрат по страхованию персонала меж-
ду подразделениями пропорционально фактическим затра-
там каждого подразделения по статье калькуляции "Основ-
ная зарплата". 
	- Использовать другой тип базы распределения. При таком 
выборе варианта расчета базы распределения для каждой 
единицы распределения задаются вручную в графе "Ба-
за".

При распределении общепроизводственных затрат дополнительно 
задаются параметры, необходимые для расчета коэффициента рас-
пределения постоянных затрат между счетом учета себестоимости 
продукции (счет 23) и счетом учета себестоимости реализованной 
готовой продукции (счет 901). 

Распределение выполняется в соответствии с пропорцией, опре-
деляемой соотношением нормальной базы распределения, нормаль-
ной суммой постоянных затрат, текущей (фактической) базой рас-
пределения и текущей (фактической) суммой постоянных затрат. 

Значения базы распределения затрат и суммы постоянных общепро-
изводственных затрат при нормальной мощности задаются в группе 
"Общепроизводственные затраты". 

Заполнить табличную части можно двумя способами: обычным 
построчным вводом или с помощью кнопки "Заполнить". После 
нажатия этой кнопки в табличную часть документа будут внесены 
все единицы указанного вида, которые фигурировали в текущем 
месяце в операциях выпуска продукции (оказания услуг) по нормам 
(т.е. зафиксированы в регистре "Нормативные затраты"). При 
распределении затрат между подразделениями, перечень отбираемых 
указанным выше образом подразделений можно ограничить:
	- только производственными подразделениями (опция 
"Только производственные");
	- только производственными и обслуживающими подразде-
лениями (опция "Включая обслуживающие");
	- или же внести полный список подразделений.

Расчет баз распределения в соответствии с выбранным способом 
осуществляется по опции "Рассчитать базы" кнопки "Рассчитать". 

Реальная база распределения, которая и будет использоваться в рас-
четах, проставляется в графе "Реальная база", которая в отличие от 
графы "База", доступна для редактирования. Автоматическое запол-
нение этой графы значением графы "База" выполняется с помощью 
опции "Скопировать базы" кнопки "Рассчитать". После этого с 
помощью опции "Привести базы к %" можно представить суммы баз 
распределения в процентом выражении.

Расчет сумм затрат, которые будут отнесены по указанным еди-
ницам распределения выполняется при проведении документа, кото-
рое можно инициировать нажатием кнопки "Провести". После этого 
суммы затрат в суммовом и процентом выражении отобразятся в 
графах "Сумма" и "Процент" соответственно.

В результате проведения документа:
	- при распределении им общепроизводственных затрат: 
формируются бухгалтерские проводки: по закрытию счета 
91 и движения в регистре "Производственные затраты", 
общая сумма которых равна итоговой сумме, списанной 
на счет со счета 91 на счет 23. 
	- при распределении затрат на брак: проводки по списанию 
сумм затрат на счет 23, движения в регистре "Производ-
ственные затраты";
	- при произвольном перераспределении (материальных и 
нематериальных затрат) - проводки по счету 23 (ДВ для 
давальческих заказов), движения по регистру "Производ-
ственные затраты": сначала по списанию полной суммы, 
подлежащей распределению "по прошлой аналитике" (ус-
тановленной в группе "Затраты"), а потом по регистрации 
сумм затрат по единицам распределения, указанным в до-
кументе. При этом остальные объекты аналитики для 
"новых" движений в регистре выбираются также из груп-
пы "Затраты" шапки документа. Исключение составляет 
лишь статья затрат, которую можно установить новой с 
помощью флажка "Отнести затраты на новый вид за-
трат".
