Возврат товарно-материальных ценностей поставщику регистри-
руется в конфигурации с помощью документа "Возврат поставщи-
ку". Этот документ удобно вводить на основании того же приходно-
го документа, с помощью которого было оформлено приобретение 
ТМЦ, подлежащих возврату.

Ввод нового документа выполняется выбором пункта "Возврат 
поставщику" в подменю "Возвраты" меню "Документы".

В группе "Поставщик" указывается контрагент-поставщик и бух-
галтерский счет учета взаиморасчетов с ним. Поскольку в типовой 
конфигурации не предусмотрен возврат импортированных товаров, 
то в качестве валюты документа (в поле "валюта") следует выбирать 
только "гривню".

Далее выбирается договор, в рамках которого были приобретены 
возвращаемые запасы и документ-счет. 

В реквизите "Место хранения" указывается тот оптовый склад, с 
которого будут списаны возвращаемые товарно-материальные цен-
ности.

В табличной части документа указывается следующая информа-
ция:
	- наименование ТМЦ, который подлежит возврату;
	- партия, которая будет списана при проведении документа 
(по учетным ценам которой происходит списание товара);
	- количество возвращаемых ТМЦ;
	- единица измерения ТМЦ;
	- коэффициент пересчета для выбранной единицы измере-
ния;
	- цена без НДС или цена с НДС (в зависимости от значе-
ния константы "Цены в накладных");
	- итоговая сумма по строке без НДС;
	- итоговая сумма по строке с НДС;
	- сумма налога на добавленную стоимость по строке. 

В графе "Ост" отображается остаток текущего ТМЦ на выбран-
ном складе на дату документа.

Для заполнения табличной части документа можно использовать 
метод множественного подбора из справочника "ТМЦ" с помощью 
кнопки "Подбор".

На закладке "Дополнительно" указывается аналитика валовых 
расходов и НДС, используемая при формировании сторнирующих 
проводок по валовым расходам и НДС. 

Здесь также можно внести информацию о доверенности (рекви-
зиты в группе "Доверенность") и о сотруднике, который отпустил 
ТМЦ (реквизит "Отпустил"), подлежащие возврату.

Реквизиты "Счет" и "Вид" задают затратный счет и его аналити-
ку для бухгалтерских проводок, которые формируются в случае, ко-
гда возврат товаров происходит по ценам дешевле их учетных цен. 

При этом в качестве вида деятельности будет выбран вид деятельно-
сти, который указан в для таких ТМЦ в справочнике "Номенклату-
ра".

Кнопка "Печать" позволяет сформировать печатную форму на-
кладной.

Кнопка "ОК" служит для записи и проведения документа.

При проведении документа по каждой строке табличной части 
формируются движения по регистру "Партии", "Остатки", "Оборо-
ты". Сторнируются имеющиеся задолженности по оплатам постав-
щику в рамках "счета" (указанного в реквизите "Счет/накладная").
