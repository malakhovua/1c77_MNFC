Для ввода начальной информации по необоротным активам пред-
приятия: по бухгалтерскому и налоговому учету (для объектов групп 
1, 2, 3, по которым должен вестись пообъектный учет) предназначен 
документ "Ввод остатков необоротных активов".

Документы "Ввод остатков необоротных активов" создаются, 
хранятся и редактируются в журнале "Документы ввода начальных 
остатков" ("Журналы"/"Ввод остатков") вместе с другими докумен-
тами ввода начальных остатков. Новый документ вызывается также 
из меню "Документы"/"Ввод остатков"/"Ввод остатков необорот-
ных активов".

В шапке документа задается вид необоротных активов и подраз-
деление, по необоротным активам которого вносятся начальные дан-
ные. Если флажок "Начисляется аморт. в налог. учете" включен, то в 
форме документа появляется группа реквизитов "Налоговый учет", 
в которой нужно определить группу, которой принадлежат необо-
ротные активы. Если по этой группе не ведется пообъектный учет, то 
становится доступным реквизит "Балансовая стоимость группы", в 
котором необходимо ввести общую балансовую стоимость всей 
группы. Напомним, что признак того, ведется или нет пообъектный 
учет в рамках той или иной группы ОС, задается для каждой группы 
в дополнительном параметре "Пообъектно" в справочнике "Шкала 
ставок".

В табличной части документа вносится следующая информация:
	- наименование необоротного актива, по которому вводятся 
начальные данные;
	- счет учета необоротного актива;
	- место хранение (МОЛ), за которым числится данный 
объект;
	- количество объектов вводится только для НМА, для объ-
ектов НА количество всегда устанавливается равным 1 и 
недоступно для редактирования;
	- первоначальная стоимость актива;
	- сумма начисленного по нему износа;
	- сумма остаточной стоимости рассчитывается автоматиче-
ски и недоступна для редактирования;
	- балансовая стоимость объекта по налоговому учету. Этот 
реквизит доступен только для групп, по которым ведется 
пообъектный учет.

Для НМА в табличной части дополнительно вводится первона-
чальная стоимость по данным налогового учета и сумма начисленно-
го износа в налоговом учете.
 
При проведении документ по бухгалтерскому учету формирует 
проводки по дебету счетов учета необоротных активов, НМА и кре-
диту счетов учета их износа. В налоговом учете для объектов групп 
1, 2, 3 формируется проводка по дебету счета ОС1, ОС2 или ОС3 на 
сумму балансовой стоимости, для НМА - по дебету/кредиту счетов 
НА1/НА2 на суммы их первоначальной стоимости и начисленного 
износа соответственно. В качестве корреспондирующего счета для 
балансовых счетов во всех перечисленных проводках фигурирует 
счет "00".
