Для выполнения распределения/перераспределения затрат произ-
водственных подразделений между объектами их учета (видами дея-
тельности, заказами, видами продукции) пропорционально нормам 
затрат предназначен документ "Распределение затрат по нормам". 

В форме документа в первую очередь указывается подразделение, 
в котором нужно распределить и уровень передела продукции, для 
которой эта операция будет выполнена.

С помощью флажков группы "Распределять…" в форме докумен-
та устанавливается тип объектов, между которыми будет произво-
диться распределение затрат. Если установлены все три флажка, то 
распределение затрат выполняется последовательно в три этапа:
	- на первом этапе распределяются все затраты отраженные 
по пустому виду деятельности между всеми видами дея-
тельности, по которым в течение месяца происходил вы-
пуск продукции. В качестве баз распределения для каждо-
го вида затрат между видами деятельности выступает ито-
говая сумма норм по статье калькуляции (в которую 
включается текущая затрата) по всем видам продукции 
заданного уровня передела. При этом нормы в нематери-
альным статьям калькуляции выбираются в суммовом, а 
по материальным -в количественно выражении;
	- на втором этапе затраты по "пустому" заказу в рамках 
"позаказного" вида деятельности распределяются между 
заказами, по которым выпускалась продукция заданного 
уровня передела (и выше) в течение месяца. Принцип оп-
ределения базы распределения полностью аналогичен 
предыдущему случаю. 
	- наконец, на третьем этапе, затраты, отраженные по "пус-
той" продукции в рамках заказов (для массового произ-
водства -по "пустому" заказу) распределяются между 
видами продукции также пропорционально нормам затрат 
по статьям калькуляции.

При необходимости можно ограничить количество этапов выпол-
нения документа одним или двумя последовательными этапами.
 
При проведении документ формирует проводки по счету 23 (ДВ 
для давальческих заказов), движения по регистру "Производствен-
ные затраты": сначала по списанию полной суммы, подлежащей рас-
пределению "по прошлой аналитике", а потом по регистрации сумм 
затрат по соответствующим единицам распределения.

этапов распределения затрат.
