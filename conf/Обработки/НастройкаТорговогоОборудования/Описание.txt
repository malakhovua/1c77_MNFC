При помощи мастера "Настройка торгового оборудования" зада-
ются различные параметры использования торгового оборудования 
на конкретном рабочем месте. 

Для доступа к этому мастеру следует в меню "Сервис" выбрать 
пункт "Торговое оборудование" / "Настройка торгового оборудова-
ния".

Все параметры, которые можно установить в мастере, расположе-
ны на шести страницах. Чтобы перемещаться по страницам мастера, 
следует пользоваться кнопками "Далее" и "Назад". На любой стра-
нице доступна кнопка "Готово", при нажатии которой подготовлен-
ные данные настройки торгового оборудования выбранного рабочего 
места записываются в компьютер. Можно отменить изменение на-
стройки торгового оборудования (кнопка "Отменить"). 

После записи настройки торгового оборудования конфигурация 
загружает и запускает необходимые драйверы торгового оборудова-
ния. Впоследствии, при запуске системы на данном рабочем месте, 
конфигурация будет при начале работы автоматически настраивать-
ся в соответствии с записанными настройками.

На первой странице мастера предлагается заполнить справочник 
ЭККА (все ЭККА компании), форма которого открывается по кноп-
ке "Справочник ЭККА".
 
Здесь для каждого кассового аппарата вводится его уникальный 
номер (код в справочнике), наименование, указывается магазин ком-
пании, путем выбора его из справочника "Склады". Также в спра-
вочнике указывается тип ЭККА (тип драйвера ЭККА) и режим ра-
боты. Тип драйвера выбирается из списка включенных в конфигура-
цию обработок обслуживания ЭККА (см. Приложение 1 "Интегра-
ция с торговым оборудованием", раздел "Подключение торгового 
оборудованиея в конфигурации"). В качестве режима работы кассо-
вого аппарата может быть установлено одно из следующих значений: 
отчет (OFF-line), регистрация (фискальный принтер), автономный 
(не подключен).

После внесения всех требуемых данных по кнопке "Закрыть" 
следует вернуться в форму мастера настройки торгового оборудова-
ния. 

На второй странице мастера предлагается выбрать рабочее место, 
для которого настраивается торговое оборудование, установить ре-
жимы использования сканера штрих-кода на данном рабочем месте: 
разрешить работу со сканером штрих-кода, тип сканера штрих-кода.
 
На третьей, четвертой и пятой страницах мастера предлагается 
заполнить список ЭККА на выбранном рабочем месте в режиме 
фискального регистратора, режиме "OFF Line" и автономном режи-
ме соответственно.
 
На шестой странице мастера можно проконтролировать настрой-
ки режимов работы кассовых аппаратов. Все ЭККА, подключенные к 
выбранному рабочему месту показываются в одной таблице, с указа-
нием режима работы ЭККА.
