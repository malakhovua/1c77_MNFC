Программа позволяет корректировать и обрабатывать 
одновременно группу элементов справочника контрагентов. 


ОТБОР ЭЛЕМЕНТОВ

В окне настройки требуется задать фильтры на элементы 
справочника, чтобы отобрать из них подлежащие групповой 
обработке. Окно настройки имеет две закладки: 
"Основная" и "Множественный фильтр". 

Все параметры фильтра (указанные на основной закладке и во 
множественном фильтре) действуют в совокупности. Поэтому к 
установке параметров в основном и множественном фильтре надо 
подходить обдуманно. 

Например, если в основном экране задан какой-то 
конкретный контрагент, а во множественном фильтре - 
группа контрагентов, в которую этот контрагент не входит, 
то после задания такого фильтра будет выдано 
сообщение "Список пуст". 

В то же время с помощью установки одновременно двух 
фильтров при правильном подборе параметров можно решить 
достаточно сложные аналитические задачи. Например, требуется 
получить список всех должников групп "Покупатели" и 
"Реализаторы" (скажем, для дальнейшего анализа и переоценки). 
Контрагентам-должникам присвоена категория "Должники". Тогда в 
фильтре на основной закладке указывается эта категория, а во 
множественном фильтре выбираются группы "Покупатели" и 
"Реализаторы".

Совет! Перед тем, как установить параметры одновременно в двух 
фильтрах, надо хорошо обдумать, какие именно данные 
желательно получить. Если задачу четко сформулировать не 
удается, то лучше пользоваться либо тем, либо другим 
фильтром. 

В случае если множественный фильтр установлен, перед 
названием закладки "Множественный фильтр" устанавливается знак 
(!). Он виден и с основной закладки.

В окне настройки групповой обработки контрагентов можно 
следующим образом задать фильтры на контрагентов, подлежащих 
групповой обработке:

Вид - выбрать из выпадающего списка вид контрагента: все 
контрагенты, только организации, только частные 
лица. 

Контрагент или группа - задать конкретного контрагента или 
группу контрагентов. С помощью множественного 
фильтра можно задать список произвольно 
выбранных контрагентов или групп контрагентов.

Категория - задать конкретную категорию контрагента. С 
помощью множественного фильтра можно задать 
список категорий контрагентов. Причем отбор 
контрагентов будет произведен по разным 
алгоритмам, в зависимости от установленного 
переключателя: "одна из" или "одновременно все". В 
первом случае будут отобраны контрагенты, 
относящиеся хотя бы к одной из указанных в списке 
категорий; во втором случае будут отобраны 
контрагенты, относящиеся сразу ко всем категориям, 
указанным в списке. 

Тип цен покупателю - задать конкретный тип цен, 
установленный по умолчанию для контрагента в 
справочнике как тип цен покупателю. С помощью 
множественного фильтра можно задать список типов 
цен. В последнем случае будут отобраны 
контрагенты, у которых хотя бы один тип из списка 
установлен по умолчанию как тип цен покупателю.

Тип цен поставщика - задать конкретный тип цен, 
установленный по умолчанию для контрагента в 
справочнике как тип цен поставщика. С помощью 
множественного фильтра можно задать список типов 
цен. В последнем случае будут отобраны 
контрагенты, у которых хотя бы один тип цен из 
списка установлен по умолчанию как тип цен 
поставщика.

Валюта взаиморасчетов - задать конкретную валюту 
взаиморасчетов, установленную у контрагента по 
умолчанию. С помощью множественного фильтра 
можно задать список валют взаиморасчетов. В 
последнем случае будут отобраны контрагенты, у 
которых установлена по умолчанию любая из 
заданных в списке валют взаиморасчетов.

Кредит покупателю - группа объектов, объединяющая параметры 
устанавливаемые для кредита покупателю:

Валюта - задать конкретную валюту кредита покупателю. С 
помощью множественного фильтра можно задать 
список валют кредита покупателю. В последнем 
случае будут отобраны все контрагенты, у которых 
валюта кредита покупателю совпадает с любой из 
списка. 

Сумма - установить интервальный фильтр по максимальным 
суммам кредита для покупателя, установленным для 
контрагента в справочнике.

Глубина кредита - установить интервальный фильтр по 
максимальной глубине кредита покупателю. 

Кредит поставщику - группа объектов, объединяющая параметры 
устанавливаемые для кредита поставщику:

Валюта - задать конкретную валюту расчетов с поставщиком, 
установленную у контрагента. С помощью 
множественного фильтра можно задать список валют 
кредита поставщику. В последнем случае будут 
отобраны все контрагенты, у которых хотя бы одна из 
заданных списком фильтра валют установлена как 
валюта кредита поставщику. 

Сумма - установить интервальный фильтр по максимальным 
суммам кредита для поставщика, установленным для 
контрагента в справочнике.

Глубина кредита - установить интервальный фильтр по 
максимальной глубине кредита поставщику. 

ВНИМАНИЕ!!! Для того чтобы получить интервальный фильтр с 
открытой границей интервала, нужно установить (или оставить) 
соответствующую границу равной нулю. Например, если задать
фильтры по глубине кредита и сумме кредита "от 5 дней до 0" 
и "от 950 долларов до 0", заданы на самом деле окажутся 
границы "от 5 дней до любой величины" и "от 950 долларов до
любой величины".

После задания фильтров и нажатия кнопки "Выбрать" на экране 
появляется окно со списком элементов справочника, отобранных для 
групповой обработки.
