Это окно настройки печати, в котором можно 
пользоваться следующими элементами управления:

Валюта печати - можно выбрать валюту, в которой будут 
выводиться все цены. Пересчет из цен, в которых эти 
цены установлены, будет осуществляться по курсам 
справочника валют на текущую дату. Если не 
устанавливать валюту печати (уже установленное 
значение можно очистить кнопкой "Х"), цены будут 
печататься для каждого товара так, как установлены. 

Единицы - выбор способа задания единиц измерения количества 
товара при печати прайс-листа из значений: "по 
умолчанию", "базовая", "задания цен". Первые два 
значения этого параметра обуславливают пересчет 
цен из единиц, для которых эти цены установлены, в 
цены для единиц выбранного типа (по 
коэффициентам пересчета соответствующих единиц). 
При выборе последнего значения цены будут 
печататься для каждого товара так, как установлены. 

Позиции и группы прайс-листа - в этом множественном 
фильтре можно задать набор позиций и групп прайс-
листа, для которых будет осуществляться вывод на 
печать. 
Если заданные позиции повторяются (например, задана 
группа и отдельно одна из позиций этой группы), 
напечатана будет одна строчка. Если не задать в этом 
фильтре ничего, печататься будет весь прайс-лист.

Типы цены - в этом множественном фильтре можно задать типы 
цен, для которых будут выводиться на печать цены. В 
отличие от предыдущего фильтра, если не задать 
ничего, никакие цены печататься не будут. Кроме 
того, порядок расположения (слева направо) колонок 
прайс-листа в печатной форме определяется 
порядком расположения (сверху вниз) строчек 
полученного списка типов цен. Строчки списка 
можно перемещать с помощью расположенных справа 
от списка кнопок со стрелками. 

Не включать отсутствующие у компании товары - установка 
этого флажка обуславливает исключение из печатной 
формы прайс-листа товаров, которых в данный 
момент нет у компании (суммарное количество 
товара на всех складах меньше или равно нулю). При 
этом из печатной формы автоматически исключаются 
услуги, которые на складах не хранятся.

При печати прайс-листа товары внутри каждой группы 
сортируются по наименованию.
