Программа позволяет корректировать и обрабатывать 
одновременно группу элементов справочника номенклатуры.

ОТБОР ЭЛЕМЕНТОВ СПРАВОЧНИКА 

Окно настройки обработки справочника. 

В окне настройки требуется задать фильтры на элементы 
справочника, чтобы отобрать из них подлежащие групповой 
обработке. Окно настройки имеет две закладки: 
"Основная" и "Множественный фильтр". 

Все параметры фильтра (указанные на основной закладке и во 
множественном фильтре) действуют в совокупности. Поэтому к 
установке параметров в основном и множественном фильтре надо 
подходить обдуманно. 

Например, если в основном экране задан какой-то конкретный 
товар, а во множественном фильтре - группа товаров, 
в которую этот товар не входит, то после задания такого фильтра 
будет выдано сообщение "Список пуст". 

В то же время с помощью установки одновременно двух 
фильтров при правильном подборе параметров можно решить 
достаточно сложные аналитические задачи. Например, требуется 
получить список всех импортных товаров групп "Обувь" и "Бытовая 
техника" (скажем, для дальнейшего анализа и переоценки). 
Импортным товарам присвоена категория "Импортные". Тогда в 
фильтре на основной закладке указывается эта категория, а во 
множественном фильтре выбираются группы "Обувь" и "Бытовая 
техника". 

Совет! Перед тем, как установить параметры одновременно в двух 
фильтрах, надо хорошо обдумать, какие именно данные 
желательно получить. Если задачу четко сформулировать не 
удается, то лучше пользоваться либо тем, либо другим 
фильтром. 

В случае если множественный фильтр установлен, перед 
названием закладки "Множественный фильтр" устанавливается 
знак (!). Он виден и с основной закладки.

В окне настройки групповой обработки справочника 
номенклатуры можно следующим образом задать фильтры на товары 
и услуги, подлежащие групповой обработке: 

Вид - выбрать из выпадающего списка вид товарной 
номенклатуры: вся номенклатура, только товары, 
только услуги, только наборы. 

Позиция или группа позиций - задать конкретный товар или 
группу товаров. С помощью множественного фильтра 
можно задать список произвольно выбранных 
товаров или групп товаров.

Категория - задать конкретную категорию товара. С помощью 
множественного фильтра можно задать список 
категорий товаров. Причем отбор товаров будет 
произведен по разным алгоритмам, в зависимости от 
установленного переключателя: "одна из" или 
"одновременно все". В первом случае будут отобраны 
товары, которые относятся хотя бы к одной из 
указанных в списке фильтра категорий; во втором 
случае будут отобраны только товары, относящиеся 
одновременно ко всем категориям, указанным в 
списке.

Единица измерения - задать конкретную единицу измерения 
количества товара. С помощью множественного 
фильтра можно задать список единиц измерения, по 
которым будет производиться отбор товаров. В 
зависимости от установленного переключателя отбор 
товаров будет производиться по-разному. Если 
установлено значение переключателя "одна из 
единиц", будут отбираться товары, среди единиц 
измерения которых есть заданная фильтром (или 
хотя бы одна из списка единиц измерения 
множественного фильтра). Если установлено 
значение переключателя "базовая", будут отбираться 
только те товары, у которых значение базовой 
единицы измерения совпадает с заданной (или с 
одной из списка). Если установлено значение 
переключателя "по умолчанию", то будут отбираться 
только те товары, у которых значение единицы 
измерения по умолчанию совпадает с заданной (или с 
одной из списка).

Валюта учета - задать конкретную валюту учета. С помощью 
множественного фильтра можно задать список валют 
учета. 

НДС - задать значение ставки НДС: любая, без НДС, 0%, 20%. 
Если задано значение "любая", то НДС в отборе не 
участвует.

ВНИМАНИЕ!!! Для того чтобы получить интервальный фильтр с 
открытой границей интервала, нужно установить (или оставить) 
соответствующую границу равной нулю. 

После задания фильтров и нажатия кнопки "Выбрать" на экране 
появляется окно со списком элементов справочника, отобранных для 
групповой обработки.
