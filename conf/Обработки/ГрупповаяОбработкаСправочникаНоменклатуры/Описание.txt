Программа позволяет корректировать и обрабатывать 
одновременно группу элементов справочника номенклатуры.

После задания фильтров и нажатия кнопки "Выбрать" на экране 
появляется окно со списком элементов справочника, отобранных для 
групповой обработки. 

Список не иерархический: в нем могут быть представлены 
элементы из всех групп справочника. Группа, к которой относится 
элемент, указана в одной из колонок списка.

В списке можно пометить какие-то из элементов с помощью 
клавиши Enter или двойного щелчка мыши, и провести необходимые 
действия одновременно со всеми помеченными элементами. Чтобы 
пометить объект, необходимо установить курсор на любое поле в 
списке, кроме поля "Пометка".

В окне списка можно также пользоваться кнопками:

"Все" - помечаются все элементы в списке,

"Снять" - снимается пометка со всех отмеченных элементов,

"Инвертировать" - снимается пометка у всех элементов, у 
которых она есть, и ставится тем, у которых ее нет.

В выпадающем списке "Действие" можно выбрать действия, 
которые будут выполнены по отношению к помеченным элементам 
справочника. Для некоторых действий можно (или 
обязательно нужно) установить дополнительные параметры 
с помощью кнопки "Значение".

Для групповой обработки товаров и услуг можно выбрать одно из 
следующих действий, которое будет произведено над всеми 
помеченными элементами списка:

Пометить на удаление - позиции будут помечены на удаление.

Снять пометку удаления - у позиций, помеченных на удаление,
будет снята эта пометка.

Перенести в группу - в поле "Значение" необходимо выбрать 
соответствующую группу справочника номенклатуры. 
Позиции будут перенесены в указанную группу. 

Установить валюту учета - в поле "Значение" необходимо 
выбрать из справочника валют валюту учета. Будет 
установлена выбранная валюта учета.

Установить ставку НДС - в поле "Значение" выбрать одно из 
значений: Без НДС, 0%, 20%. Будет установлена 
выбранная ставка НДС.

Установить категорию - в поле "Значение" выбрать категорию 
из справочника категорий. Если такой категории не 
было в списке категорий по данной позиции, то она 
добавится (товар может относиться к нескольким 
категориям одновременно).

Удалить из категории - в поле "Значение" выбрать категорию 
из справочника категорий. Если такая категория была 
в списке категорий по данной позиции, то она будет 
из него удалена.

Удалить из всех категорий - в этом случае удалятся все 
категории из списка категорий по всем позициям.

Добавить единицу измерения - в поле "Значение" выбрать из 
справочника единиц единицу измерения количества 
товара. Если какая-либо из обрабатываемых позиций 
не имела соответствующей единицы измерения, будет 
добавлена новая единица измерения с пустым 
штрихкодом.

Удалить единицу измерения - в поле "Значение" выбрать из 
справочника типов единиц тип единицы измерения. 
Если какая-либо из обрабатываемых позиций имела 
заданную единицу измерения, и эта единица отлична 
от базовой, то она будет помечена на удаление. Если 
при этом помечаемая на удаление единица совпадает 
с единицей по умолчанию, то единица по умолчанию 
устанавливается равной базовой единице.

Установить комментарий - установить примечание во всех 
отобранных позициях. Текст вводится в окне, 
появляющимся при нажатии кнопки "Значение". 
Установить мин. тов. остаток - в поле "Значение" ввести число, 
равное минимальному товарному остатку. У позиций 
установится минимальный товарный остаток, равный 
заданному.

Установить торг. наценку - в поле "Значение" ввести число, 
равное проценту торговой наценки. У позиций 
установится процент торговой наценки, равной 
заданному.

По кнопкам:

"Выполнить" - выполняется обработка помеченных элементов 
справочника,

"Закрыть" - выход без обработки в окно настройки.
