Каждая номенклатурная единица характеризуется базовой единицей 
измерения. Это единица, в которой ведется учет наличия товара.
Кроме базовой единицы измерения, товар может иметь произвольное 
количество других единиц измерения (упаковок). Например, сигареты 
могут иметь следующие единицы измерения: штука, пачка, блок, коробка, 
контейнер, вагон, состав. Общий список применяемых компанией единиц 
измерения заносится в справочник "Единицы измерения". Для просмотра 
и редактирования справочник можно открыть из меню "Справочники" 
выбором пункта "Номенклатура" - "Единицы измерения". 

Каждая единица измерения ТМЦ должна иметь свой коэффициент 
пересчета относительно базовой единицы измерения и свой штрих-код. 

Настоящая конфигурация позволяет указывать ТМЦ в документах в 
любой единице измерения из числа введенных для данного ТМЦ. Одна из 
единиц измерения товара определяется в форме товара, как единица 
измерения по умолчанию (в одноименном реквизите на закладке 
"Единицы"). Единица измерения по умолчанию используется при 
подборе товаров в документ и при выводе данных по товару в отчетах.

Для хранения данных о единицах измерения товара в конфигурации 
существует справочник "Единицы товара", подчиненный справочнику 
"Номенклатура". Ввод и редактирование данных о единицах измерения 
конкретного товара можно выполнить непосредственно на закладке 
"Единицы". Здесь представлен список единиц измерения товара и кнопки 
для управления этим списком. Для каждой единицы измерения 
отображается наименование, коэффициент пересчета, штрих-кода 
единицы измерения.  

С помощью кнопки "Добавить" открывается диалог "Сведения о 
единице товара" для ввода информации о новой единице измерения 
товара.        

Единица измерения товара может иметь свой собственный штрих-код. 
Занесение штрих-кода единицы измерения товара может быть выполнено 
либо вручную, либо непосредственно с помощью сканера штрих-кода. 
Можно также автоматически сформировать уникальный штрих-код, 
нажав специальную кнопку. 

Кнопка "Изменить" открывает для редактирования заполненный 
диалог с информацией о единице измерения, выбранной в списке.

Нажатие кнопки "Удалить" помечает на удаление текущую единицу 
измерения товара в справочнике "Единицы измерения" и удаляет 
единицу измерения из списка единиц измерения настоящей закладки.
Для отмены пометки на удаление единицы измерения следует из 
справочника "Номенклатура" открыть подчиненный справочник 
"Единицы" и в форме списка снять отметку на удаление.
