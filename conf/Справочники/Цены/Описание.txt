Для каждого товара может быть назначено несколько цен - 
столько, сколько типов цен введено в справочнике "Типы цен". При 
назначении товару цены данного типа можно выбрать, в какой 
валюте она будет измеряться, и за какую единицу измерения 
количества товара (упаковку) будет эта цена.

Новые цены на товар могут назначаться вручную, путем записи 
информации в справочник цен по конкретному товару в его форме 
или формироваться в полуавтоматическом режиме с помощью 
групповой обработки ("Сервис", "Формирование цен на товары").
Кроме того, цены на товар могут быть переписаны из 
документов, сопровождающих получение товара (кнопка "Действия",
"Обновление цен в справочнике"). 
Также они могут быть переписаны из внешнего файла ("Сервис",
"Импорт справочника цен"). 
Наконец, они могут быть сформированы в документе "Акт 
переоценки").

Можно вводить новые цены на товар и редактировать имеющиеся 
в подчиненном справочнике цен, вызываемом из каталога товаров, 
или - для каждого товара - на закладке "Цены" формы "Сведения о 
товаре". 
При обоих способах для того, чтобы ввести новую цену на товар 
необходимо проделать следующие действия:

1. В справочнике цен по товарам (непосредственно или на 
закладке "Цены") добавить новую запись. При редактировании в 
диалоге будет вызвана форма "Сведения о цене".

2. Выбрать тип цены. Тип цены выбирается из выпадающего 
списка, сформированного на основании справочника типов цен.

3. Выбрать единицы измерения количества и цены товара 
(упаковку и валюту).

4. Ввести наценку. По умолчанию устанавливается суммарная 
наценка, складывающаяся из наценки, указанной в справочнике 
типов цен для выбранного типа цены, и торговой наценки данного 
товара, указанной на основной закладке формы товара. Наценку 
можно отредактировать вручную. Может быть назначена 
индивидуальная наценка для каждой цены каждого товара.

5. Ввести значение цены.

6. Доступ к истории цен осуществляется по кнопке "История".
