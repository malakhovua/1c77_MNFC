Контрагент может иметь несколько расчетных счетов для 
осуществления денежных расчетов. Для хранения информации о 
банковских счетах контрагента в конфигурации предусмотрен 
справочник "Денежные счета", подчиненный справочнику 
"Контрагенты". 

Для удобства работы с этим справочником ввод и редактирование 
информации о расчетных счетах контрагента может осуществляться с 
помощью управляющих элементов закладки "Расчетные счета". Закладка 
"Расчетные счета" появляется в форме справочника только после того, 
как вы записали новый элемент справочника "Контрагенты", и открыли 
его форму для редактирования.

Кнопка "Добавить" открывает пустой диалог для ввода информации 
о новом счете. Для каждого элемента справочника должно быть 
определено наименование, т. е. "внутреннее" название банковского счета 
контрагента ("главный", "дополнительный", "валютный" и т.д.), валюта, 
в которой учитываются средства на данном счете, а также его банковские 
реквизиты. Банк, в котором открыт расчетный счет указывается выбором 
из справочника "Банки".   

Кнопка "Изменить" открывает для редактирования заполненный 
диалог с информацией о расчетном счете, выбранном в списке.     

Кнопка "Удалить" удаляет счет из списка счетов закладки 
"Расчетные счета", но только помечает соответствующий элемент в 
справочнике "Денежные счета".
