Список наименований выпускаемой продукции, полуфабрикатов, 
материалов, товаров для перепродажи, хранится в справочнике "Но-
менклатура". Кроме этого, в этом справочнике могут храниться на-
именования оказываемых и приобретаемых услуг, а также описания 
наборов товаров.

Он используется как при заполнении документов, так и для веде-
ния аналитического учета по бухгалтерским счетам.

Для удобства поиска информации справочник реализован трех-
уровневым, т.е. номенклатуру продукции, товаров, услуг и пр. в нем 
можно объединять в произвольные группы и подгруппы. Кроме того, 
с помощью присвоения категорий (аналитических признаков) реали-
зована возможность дополнительной классификации номенклатур-
ных единиц. Одному ТМЦ можно присвоить несколько категорий. В 
отчетах можно детализировать информацию до уровня категорий 
ТМЦ.

Для каждого ТМЦ может быть также назначено несколько типов 
цен. Названия цен задаются в справочнике "Типы цен". Новый тип 
цен может быть введен непосредственно при заполнении новой цены 
в форме ТМЦ. 

Сами цены могут быть либо занесены непосредственно в форму 
номенклатурной единицы, либо сформированы с помощью механиз-
ма групповой обработки цен на основании учетной цены и введен-
ной по каждой цене ТМЦ торговой наценки. (Подробнее о механиз-
ме ценообразования см. раздел "Сервисные возможности при работе 
со справочниками").

Каталог ТМЦ можно просмотреть и распечатать. Для этого сле-
дует воспользоваться кнопкой "Печать...", расположенной в окне 
списка товаров, и выбрать пункт "Каталог" из выпадающего меню. 
После этого перед вами раскроется диалог настройки параметров 
печати каталога, форма которого в точности совпадает с формой за-
проса параметров печати прайс листа.

С помощью пунктов меню "Ценник" и "Этикетка" для каждой 
номенклатурной единицы можно сформировать и распечатать цен-
ник и этикетку со штрих-кодом. 

Для каждого ТМЦ можно открыть подчиненный справочник цен 
с помощью кнопки "Цены".

Кнопкой "Групповая обработка" вызывается окно подготовки 
групповой обработки элементов справочника.

Все сведения о каждой номенклатурной единице распределены по 
нескольким закладкам диалога: "Основные", "Параметры учета", 
"Цены", "Единицы", "Категории", "Набор" (только для ТМЦ, 
имеющего вид "Набор"), "Продукция" (только для ТМЦ, имеющих 
вид "Продукция", "Полуфабрикат" или "Услуга").

Ввод информации о новой номенклатурной единице следует на-
чинать с указания ее основных характеристик на закладке "Основ-
ные".

Каждая номенклатурная единица имеет краткое и полное наиме-
нование - краткое служит для отображения ее в экранных формах и 
для быстрого поиска, а полное наименование выводится в печатные 
формы документов.

Поле "Вид ТМЦ" может принимать одно из следующих значений - 
"Товар", "Услуга", "Тара", "Продукция", "Материал", "Стройма-
териал", "Топливо", "Запчасти", "МБП", "Полуфабрикат" или "На-
бор". 

Отметим особенности учета каждого из перечисленных видов ТМЦ:
	- "Товар". Под товаром понимается ТМЦ, который приоб-
ретается организацией для последующей продажи. Бух-
галтерский счет учета товаров по умолчанию - 281.
	- "Услуга". Под услугой понимаются как услуги, оказывае-
мые самой организацией, так и услуги, приходуемые от 
других организаций. Для услуги в форме справочника 
доступен флажок "Транспортная затрата", при установке
которого в "Налоговой накладной" будет автоматически 
заполняться соответствующая сумма. Оказываемые 
организацией услуги подразделяются на калькулируемые
и некалькулируемые. Признаком калькулируемости услуги 
является наличие норм на выпуск такой услуги по указанному 
в документе заказу. Для калькулируемой услуги учет затрат 
и расчет себестоимости ведется так же, как и для продукции. 
Для учета некалькулируемой услуги используется упрощенная 
схема, без использования 23 счета и окончательного расчета 
себестоимости единицы услуги. Бухгалтерский счет учета 
услуги по умолчанию - 92. Для оказываемых организацией 
услуг этот счет будет использоваться в проводках при 
возврате услуги. Исключение составляет 231 счет, для 
которого при возврате будет использоваться счет 24. 
Отметим также, что транспортно-заготовительными расходами 
считаются услуги со счетом учета 2092 или 289. 
	- "Тара". Особенности учета тары, в зависимости от ее вида 
(оборотная, возвратная, залоговая) приведено в разделе 
"Учет тары". Счет учета тары по умолчанию - 284.
	- "Продукция". Под продукцией понимается ТМЦ, кото-
рый производится и продается организацией. Счет учета 
продукции по умолчанию - 26.
	- "Материал". Под материалом понимается ТМЦ, который 
приобретается организацией и используется затем для 
производства продукции. Бухгалтерский счет учета мате-
риалов по умолчанию - 201.
	- "Стройматериал", "Топливо". Под этими видами ТМЦ 
понимаются различные стройматериалы и топливо. Бух-
галтерские счета по умолчанию - 205 и 203 соответствен-
но.
	- "Запчасти". К этим видам ТМЦ относятся запасные части 
и агрегаты, предназначенные для ремонта и замены изно-
шенных машин, транспортных средств, различного обору-
дования. Бухгалтерский счет учета 207.
	- "МБП". Малоценные и быстроизнашивающиеся предме-
ты (мелкое оборудование, инвентарь, спецодежда и пр.), 
срок эксплуатации которых меньше одного года. Счет 
учета МБП -22.
	- "Полуфабрикаты". Полуфабрикаты могут производиться 
и продаваться организацией. Бухгалтерский счет учета 25.
	- "Набор" имеет те же реквизиты, что и "Товар" и еще до-
полнительные реквизиты, вынесенные на закладку "На-
бор". 

Единица измерения, в которой хранится информация об остатках номенклатуры
задается в реквизите "Базовая единица измерения". В эту единицу 
пересчитываются все прочие единицы измерения количества данного 
товара (упаковки) с помощью коэффициентов пересчета. Базовую 
единицу можно выбрать из предлагаемого списка, созданного из 
справочника единиц измерения. Список принятых для товара единиц 
измерения формируется в подчиненном справочнике "Единицы 
измерения".

Единица измерения, в которой мы хотим видеть остатки ТМЦ 
в различных отчетах конфигурации, задается в реквизите
"по умолчанию" на закладке "Единицы".
В реквизите "Ставка НДС" выбирается ставка НДС, которой об-
лагается данный ТМЦ.

Если в реквизите "Артикул" ввести артикул товара, то по нему 
можно будет выполнять быстрый поиск и сортировку.

Такие параметры ТМЦ как его тип (штучный, весовой) и вес но-
сят справочный характер и задаются в реквизитах "Тип" и "Вес" 
соответственно.

С помощью специальной кнопки "Штрих-код" можно автомати-
чески сформировать уникальный штрих-код товара.

В реквизите "Минимальный остаток" указывается тот минималь-
ный запас ТМЦ, который необходимо постоянно иметь на складе. 
Этот реквизит используется при формировании отчета по остаткам 
ТМЦ, в частности, можно сформировать отчет по таким ТМЦ, запас 
которых меньше критического (или равен ему).

Ввод данных, необходимых для бухгалтерского учета номенкла-
турной единицы осуществляется на закладке "Параметры учета". 
Здесь для каждого ТМЦ должен быть указан бухгалтерский счет 
учета, вид деятельности и вид затрат. Эта информация будет исполь-
зоваться при формировании проводок по приходованию и списанию 
ТМЦ.

Каждый ТМЦ имеет свою учетную цену, которая задается в на-
циональной валюте (гривнях) без учета НДС, и процент торговой 
наценки. Эти параметры задаются на закладке "Цены". 

Учетная цена- справочная величина, которая используется для 
расчета всех остальных цен товара. 

Кроме учетной цены, каждый ТМЦ может иметь несколько цен - 
столько, сколько типов цен введено в справочнике "Типы цен". При 
назначении ТМЦ цены того или иного типа следует установить, в 
какой валюте она будет измеряться, за какую единицу измерения 
количества товара (упаковку) будет эта цена и процент наценки.

Новые цены на ТМЦ могут назначаться вручную, путем записи 
информации в справочник цен по конкретному товару в его форме 
или формироваться в полуавтоматическом режиме с помощью груп-
повой обработки.

Каждая номенклатурная единица характеризуется базовой едини-
цей измерения. Это единица, в которой ведется учет наличия ТМЦ.
Кроме базовой единицы измерения, каждый ТМЦ может иметь 
произвольное количество других единиц измерения (упаковок). Об-
щий список применяемых компанией единиц измерения заносится в 
справочник "Единицы измерения". Ввод и редактирование 
данных о единицах измерения конкретного ТМЦ можно выполнить 
непосредственно на закладке "Единицы". Здесь представлен список 
единиц измерения ТМЦ и кнопки для управления этим списком. 

Для каждой единицы измерения отображается наименование, коэф-
фициент пересчета, штрих-кода единицы измерения.

С помощью кнопки "Добавить" открывается диалог "Сведения о 
единице ТМЦ" для ввода информации о новой единице измерения 
ТМЦ. Кнопка "Изменить" открывает для редактирования заполненный 
диалог с информацией о единице измерения, выбранной в списке.
Нажатие кнопки "Удалить" помечает на удаление текущую еди-
ницу измерения ТМЦ в справочнике "Единицы измерения" и уда-
ляет единицу измерения из списка единиц измерения текущей 
закладки.

Для каждого ТМЦ может быть введено множество аналитических 
признаков (категорий), причем один и тот же ТМЦ может одновре-
менно относиться к нескольким категориям.

Категории для ТМЦ задаются путем выбора из справочника "Ка-
тегории". Удаление категорий для ТМЦ производится по кнопке 
"Удалить".

После записи элемента справочника вида "Продукция", "Полу-
фабрикат" или "Услуга" в его диалоговой форме появляется заклад-
ка "Продукция". Здесь для каждого вида продукции должен быть 
задана глубина ее передела и график формирования ее партий. 
 
В многострочной части закладки в дальнейшем будут представле-
ны нормы расходов материалов, технологических операций и узлов, 
необходимых для производства данного вида продукции (полуфаб-
рикатов, услуг). Нормы расходов для каждого вида продукции зада-
ются в конфигурации с помощью документа "Нормы расходов", ко-
торый в момент внесения в справочник "Номенклатура" нового вида 
продукции отсутствует в информационной базе. Вызвать форму та-
кого документа можно, нажав на кнопку "+" на текущей закладке 
справочника. Порядок задания норм подробно рассмотрен в разделе 
"Учет прямых производственных затрат и выпуск продукции по 
нормам".

График, т.е. частота формирования новых партий готовой про-
дукции выбирается из справочника "Графики формирования пар-
тий", форма которого открывается по кнопке "…". Если требуемый 
график еще отсутствует в этом справочнике, то внести его туда мож-
но именно на данном этапе. Каждый график характеризуется наиме-
нованием и частотой, которая может быть: ежемесячной, ежедневной 
или произвольной.

В конфигурации реализована возможность продажи наборов. При 
этом оформлять продажу наборов ТМЦ можно только торговыми 
расходными накладными. 

При повторном открытии формы редактирования характеристик 
ТМЦ вида "Набор" в ней появляется закладка "Набор". 

С помощью кнопки "Добавить" открывается диалог "Сведения о 
единице комплектации" для ввода информации о новой составляю-
щей набора. Кнопка "Изменить" открывает для редактирования 
диалог с информацией о составляющей, выбранной в списке. 
Нажатие кнопки "Удалить" удаляет текущую составляющую из 
списка закладки "Набор".
