Справочник "Шкала ставок налогов и отчислений" содержит 
список налогов, льгот, отчислений, сборов, платежей в бюджет и в 
различные государственные целевые фонды. Здесь содержится также 
перечень возможных льгот организации по отчислениям в фонды, 
льгот, по исчислению подоходного налога, начислению страховых 
взносов в Фонд социального страхования на случай безработицы, 
шкалы оплаты больничного, доплаты за выслугу лет и пр.

Справочник используется для автоматического расчета сумм на-
логов и отчислений, при редактировании документов, а также для 
ведения аналитического учета по бухгалтерским счетам: 17, 54, 6413, 
6414, 6415, 6416, 6417, 6418, 6422, 6423, 6424, 6426, 651, 652, 653.

Каждый элемент справочника характеризуется уникальным тек-
стовым кодом и набором реквизитов.
 
В периодическом реквизите "Ставка" для каждого из налогов (а 
также для льгот по отчислениям в фонды) указывается ставка налога 
(отчисления) в долях единицы.

Для льгот по подоходному налогу в этом реквизите указывается 
параметр, который определяет способ расчета подоходного налога по 
той или иной льготе. 

Периодический реквизит "Предел" служит для организации шка-
лы налогообложения для тех налогов, которые имеют несколько раз-
личных ставок в зависимости от базы налогообложения, например, 
подоходный налог. В этом реквизите указывается предел базы нало-
гообложения, до которого действует указанная ставка (или, к приме-
ру, количество лет стажа до которого действует процент оплаты 
больничного).

Для каждого налога, начисление которого производится по шкале 
налогообложения, в справочнике создана отдельная группа или под-
группа. Элементы такой подгруппы содержат значения базы предела 
и соответствующую этому пределу ставку налога.

Наименования элементов такой группы следующие: "1 предел", 
"2 предел" и т.д. в порядке возрастания предела шкалы.
В реквизите "Вид затрат" из одноименного справочника  указы-
вается требуемая статья затрат, по которой нужно отражать сумму 
текущего налога/отчисления. При этом вид затрат можно задать как 
для элемента так и для группы справочника.

В периодическом реквизите "Дополнительно" хранятся любые 
дополнительные параметры, которые нужны для правильного расче-
та, в виде строки:
<Параметр1> = <Значение1>; <Параметр2> = <Значение2>;

Такой механизм используется для хранения параметров, которые 
в последствии могут не понадобиться (в связи с изменениями в за-
конодательстве) и которые имеют смысл только для какого-либо 
конкретного налога.

Для удобного редактирования списка параметров и их значений в 
окнах редактирования элементов и групп справочника предусмотре-
ны кнопки "Новый" и "Удалить".

Например, максимальная база налогообложения при расчете от-
числений в Пенсионный фонд равна на текущий момент 1600 грн. 
Это специфический параметр, который необходим только при расче-
те отчислений в Пенсионный фонд и который в дальнейшем, воз-
можно, вовсе не будет учитываться. В списке параметров он может 
быть задан так Макс. база = 1600; .

Помимо вышеперечисленных реквизитов для каждого элемента 
справочника могут быть указаны:
	- балансовый счет, на котором учитываются расчеты по 
данному налогу или отчислению;
	- платежные реквизиты для перечисления налогов и отчис-
лений: наименование организации-получателя, ее расчет-
ный счет, наименование и код платежа.

По кнопке "История" можно просмотреть историю изменения 
значений периодических реквизитов конкретного элемента (ставки, 
предела, дополнительных параметров).

В начальной поставке конфигурации данный справочник уже за-
полнен и не требует внесения дополнительных элементов пользова-
телем. Коррекция информации в справочнике необходима только в 
случае изменения шкалы/ставок налогообложения.
