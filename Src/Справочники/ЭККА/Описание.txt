 Настоящий справочник предназначен для хранения 
 списка используемых для розничной торговли 
 контрольно-кассовых аппаратов.
 
Для каждого кассового аппарата вводится его уникальный 
номер (код в справочнике), наименование, указывается 
магазин компании, путем выбора его из справочника "Склады". 
Также в справочнике указывается тип ЭККА (тип драйвера ЭККА)
и режим работы. 

Тип драйвера выбирается из списка включенных в конфигурацию 
обработок обслуживания ЭККА.

 В качестве режима работы кассового аппарата может быть 
 установлено одно из следующих значений: 
 	- отчет (OFF-line); 
 	- регистрация (фискальный принтер);
 	- автономный (не подключен).
    
Заполнять этот справочник удобно с помощью специально 
предусмотренного для этого мастера настройки торгового оборудования
(см. описание мастера "Настройка торгового оборудования").
