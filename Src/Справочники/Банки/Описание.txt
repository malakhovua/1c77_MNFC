Справочник "Банки" предназначен для хранения списка банков 
Украины, которые используются при указании банковских реквизи-
тов фирм, входящих в состав компании, и банковских реквизитов 
контрагентов, хранящихся в справочнике "Контрагенты".
 
Информация из этого справочника используется для подстановки 
банковских реквизитов в печатные формы документов.
В качестве кода элемента справочника во внутренней классифи-
кации выступает код МФО банка. Для каждого элемента справочни-
ка вводится наименование, его адрес и код по ЕГРПОУ. Реквизит 
"Примечание" служит для внесения какой-либо дополнительной 
информации.
