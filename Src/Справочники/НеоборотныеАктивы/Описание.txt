В справочнике "Необоротные активы" ведется список как объек-
тов ОС, НМА, других необоротных материальных активов. Посколь-
ку справочник имеет трехуровневую структуру, то именно такие 
группы и рекомендуется в нем создать.

Справочник используется также для ведения аналитического уче-
та по бухгалтерским счетам: 10 "Основные средства", 11"Другие 
необоротные материальные активы", 12 "Нематериальные активы",
13 "Износ необоротных материальных активов", НА "Балансовая
стоимость НМА (налоговый учет)", ОС1 "Основные средства, 
группа 1 (налоговый учет)", ОС2 "Основные средства, группа 2 
(налоговый учет)", ОС3 "Основные средства, группа 3 (налоговый 
учет)", 01"Арендованные необоротные активы".

Каждый элемент справочника характеризуется, прежде всего, ви-
дом, который задается в поле "Вид необор. актива". Вид необоротно-
го актива однозначно определяет счет и методологию его учета и 
может принимать следующие значения: "Осн. средства", "Другие 
необоротные материальные активы", "Нематериальные активы".

Для каждого необоротного актива следует также ввести следую-
щие сведения: полное наименование, первоначальную стоимость, ин-
вентарный номер. В реквизите "Дата ввода в эксплуатацию" указы-
вается дата зачисления необоротного актива на баланс организации. 
Значение этого реквизита справочника может быть заполнено при 
проведении документа "Ввод в эксплуатацию".
 
В отличие от справочника "Номенклатура", для элементов кото-
рого можно использовать несколько единиц измерения, для элемен-
тов настоящего справочника существует только одна единица изме-
рения (реквизит "единица измерения"). Счет учета необоротного 
актива автоматически заполнится после выбора вида необоротного 
актива (в реквизите "Вид необорот. актива"). 

Реквизит "Состояние" характеризует балансовое состояние не-
оборотного актива, а именно, находится ли объект: 
	- в эксплуатации
	- на консервации;
	- в ремонте или на модернизации;
	- в оперативной аренде;
	- списан с баланса вследствие ликвидации или продажи;
	- прочее.

При оформлении ввода в эксплуатацию объектов НА, в том числе 
после модернизации, реквизит "Состояние" заполняется соответст-
вующим значением автоматически (при проведении документов 
"Ввод в эксплуатацию" и "Модернизация НА", "Ликвидация НА"). 

В дальнейшем изменять состояние объекта НА рекомендуется с 
помощью документа "Изменение параметров НА".

Параметры начисления износа необоротных активов по бухгал-
терскому и налоговому учету задаются на закладке "Дополнитель-
но".
 
Для объектов основных средств начисление амортизации осуще-
ствляется с использованием следующих методов (варианты значения 
реквизита "Метод расчета износа):
	- прямолинейного списания;
	- уменьшения остаточной стоимости (уменьшения остатка);
	- ускоренного уменьшения остатка;
	- кумулятивного;
	- производственного;
	- налогового, когда будут применены нормы и методы на-
числения амортизации, предусмотренные налоговым за-
конодательством.

Для определения амортизируемой стоимости объекта должна 
быть указана его ликвидационная стоимость (в реквизите "Ликвида-
ционная стоимость").

Далее, в зависимости от выбранного способа начисления аморти-
зации, вводятся дополнительные параметры, необходимые для его 
расчета.

Так, при прямолинейном и кумулятивном методах списания сле-
дует указать срок полезного использования объекта (в месяцах).

Эта же величина должна быть определена для методов уменьше-
ния остатка и ускоренного уменьшения остатка. Годовая норма 
амортизации для этих методов будет определена при первом начис-
лении износа по таким объектам, и записана в реквизит "Годовая 
норма амортизации". При этом данная величина недоступна для ре-
дактирования.

При выбранном способе начисления амортизации пропорцио-
нально объему продукции (работ) предусматривается заполнение 
двух реквизитов.

В реквизите "Расчетный объем производства" указывается общий 
объем продукции (работ), который может быть произведен или вы-
полнен при использовании данного объекта основных средств.

Фактический ежемесячный объем произведенной продукции ука-
зывается в реквизите "Объем производства текущего месяца". Этот 
периодический реквизит следует заполнять за каждый месяц отдель-
но. В отличие от большинства периодических реквизитов, новые 
значения которых "вступают в силу", начиная с даты их изменения, 
в реквизите "Объем производства текущего месяца" указывается 
значение, которое "действовало" в течение прошедшего месяца. 

	Например, за февраль объем продукции составил 1500 единиц. В 
	этом случае новое значение 1500 устанавливается с датой 28.02, но 
	именно это значение используется в документе "Начисление изно-
	са". За март объем продукции составил 1800 единиц. Новое значение 
	1800 нужно ввести датой 31.03. При начислении амортизации за 
	март будет использовано значение реквизита 1800, а не 1500.

Если в реквизите "Метод расчета износа" выбрано значение "на-
логовый", то это значит, что для данного объекта будет использо-
ваться метод начисления амортизации, предусмотренный налоговым 
законодательством.

С помощью опций в группе "Налоговый учет" нужно определить 
группу, которой принадлежит такой объект основных средств, что в 
свою очередь предопределит способ начисления его износа. Все па-
раметры, необходимые для начисления износа в налоговом учете 
заданы группе "Ставки, связанные с ОС, НМА, МБП" справочника 
"Шкала ставок". Для элементов "Амортизация группы 1", "Аморти-
зация группы 2", "Амортизация группы 3" здесь введен дополни-
тельный параметр "Пообъектно", значение "1" которого является 
признаком того, что в соответствующей группе ОС ведется пообъ-
ектный учет.

Для основных средств, которые принадлежат, согласно принятой 
классификации, группе других необоротных активов, амортизация 
может начисляться либо с использованием прямолинейного метода, 
либо производственного.

Метод амортизации нематериального актива выбирается, исходя 
из условий получения будущих экономических выгод. Если же такие 
условия определить затруднительно, то амортизация начисляется с 
использованием прямолинейного метода.

Амортизация малоценных необоротных активов (счет учета 111 
или 112) может начисляться:
	- в размере 50% амортизируемой стоимости объекта в мо-
мент его ввода в эксплуатацию, в размере оставшихся 50% 
стоимости - в месяце его списания с баланса (ликвида-
ции). В этом случае в реквизите "Метод расчета износа" 
нужно выбрать "50%-50%";
	- в размере 100% стоимости объекта в момент его ввода в 
эксплуатацию. Этому методу расчета износа соответствует 
значение "100%".

Включенный флажок "Начисление износа в НУ" указывает на 
необходимость начисления износа по необоротному активу в налого-
вом учете.

В группе реквизитов "Налоговый учет" для такого объекта долж-
на быть дополнительно определена группа ОФ, которой он принад-
лежит, с целью определения соответствующего субсчета забалансо-
вого счета ОС "Балансовая стоимость ОС (налоговый учет)", на ко-
тором будет вестись учет балансовой стоимости этого объекта по 
налоговому учету (см. подробнее об этом раздел "Ввод в эксплуата-
цию").

Счет затрат, вид деятельности и аналитика (вид) затрат, на кото-
рый будут отнесены суммы начисленного износа по необоротному 
активу задаются в группе "Затратная аналитика по амортизации". 

Для производственных необоротных активов, расходы по амортиза-
ции которых относятся на счет 23, можно также указать в конкрет-
ную продукцию, которая производится при помощи этого объекта. 

Если последний участвует в производстве продукции по нескольким 
видам деятлеьности, то реквизит "Вид деятельности", который за-
полнен значением константы "Вид деятельности по умолчанию" 
следует очистить. Это действие выполняется с помощью опции 
"Очистить значение" меню "Действие" главного меню программы 
или с помощью комбинации клавиш Shift+F4. Аналогично, если с 
помощью объекта  НА производится нескольких номенклатурных 
единиц продукции, то реквизит "Продукция" нужно оставить не за-
полненным. Распределение суммы начисленного износа по такому 
объекту между соответствующими видами деятельности и видами 
продукции выполняется с помощью документов "Распределение за-
трат", "Распределение затрат по нормам".

В дальнейшем изменять такие периодические параметры необо-
ротного актива как: счет учета, состояние, метод расчет износа, счет 
затрат, вид деятельности и вид затрат, параметры, необходимые для 
применения того или иного метода рекомендуется только с помощью 
документа "Изменение параметров НА".

С помощью кнопки "Печать" можно сформировать печатную 
форму инвентарной карточки необортного актива по типовой форме 
№ 03-6 (опция "Типовая форма 03-6") или в произвольной форме 
(опция "Произвольная форма").
