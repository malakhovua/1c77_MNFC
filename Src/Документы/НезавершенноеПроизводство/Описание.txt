Для ввода в информационную базу данных о незавершенном 
производстве в типовой конфигурации предназначен документ "Не-
завершенное производство". Под незавершенным производством в 
конфигурации понимается продукция, составляющие на которую 
списаны в полном объеме, но технологический процесс ее производ-
ства пока не завершен.

Ввод нового документа выполняется выбором пункта "Незавер-
шенное производство" из подменю "Затраты" меню "Документы" 
главного меню программы.

В шапке документа указывается подразделение, по которому вво-
дится информация о незавершенном производстве и статья кальку-
ляции, на основании которой определяется коэффициент завершен-
ности работ. Если этот реквизит оставить незаполненным, то  это 
интерпретируется как то, что по текущей продукции в производство 
только были списаны материалы, но никаких работ по ней не вы-
полнялось.
 
Данные о суммах незавершенного производства вносятся в таб-
личную часть документа. Здесь должно быть указано:
	- наименование продукции, по которой указываются остат-
ки в незавершенном производстве;
	- вид деятельности;
	- заказ на производство продукции для позаказного произ-
водства продукции;
	- количество единиц продукции;
	- единица измерения;
	- коэффициент пересчета для выбранной единицы измере-
ния;
	- коэффициент завершенности, на основании которого 
можно судить о том, какую долю всей стадии обработки 
"прошла" продукция.

Расчет коэффициента незавершенности, который по умолчанию 
устанавливается равным 1, выполняется по кнопке "Рассчитать": 

Он определяется как отношение суммы фактических затрат, списан-
ных в рамках указанного заказа по выбранной статье, к норме таких 
затрат. Если заказ не указан, то норма затрат по статье выбирается 
из справочника "Номенклатура".
