Для отражения в учете операций, связанных со списанием в про-
изводство технологических операций и одновременным начислением 
заработной платы сотруднику, выполнившему эти операции, предна-
значен документ "Сдельный наряд". Наряд вводится на одного рабо-
чего и представляет собой список работ, выбираемых из справочника 
"Работы".

В шапке документа указывается сотрудник, выполнивший рабо-
ты, подразделение, в котором эти работы проводились, и вид затрат.
 
В табличной части перечисляется список выполненных сотрудни-
ком работ и их количество. Начисление оплаты по сдельному наряду 
осуществляется суммированием расценки за работу, умноженной на 
количество выполненных работ.

Дополнительно для каждого вида работ может быть указано:
	- вид деятельности;
	- заказ, по которому производилась такая продукция.
	- продукция, для производства которой выполнялась рабо-
та.

Если хотя бы один из указанных параметров не определен, то для 
отнесения стоимости определенной работы в состав производствен-
ных затрат по соответствующим видам деятельности (заказам, про-
дукции) в этом случае нужно будет воспользоваться соответствую-
щими документами конфигурации. 

При проведении документа формируются проводки по списанию 
начисленных сотруднику сумм на счет 23, движения по регистру 
"Производственные затраты". Если в качестве заказа был выбран 
заказ на производство давальческой продукции, то дополнительно 
формируется проводка по дебету счета ДВ.
