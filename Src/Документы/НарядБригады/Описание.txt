Для внесения информации о выполненной бригадой сдельной ра-
боте, и распределения суммы заработка по наряду между членами 
бригады предназначен документ "Бригадный наряд". 

Исходными данными для расчета любого наряда являются сумма 
фонда заработной платы, выданной на наряд, отработанные каждым 
сотрудником бригады часы, размер почасового тарифа и коэффици-
ент трудового участия каждого работника. Опции документа "Учи-
тывать КТУ", "Учитывать часы", "Учитывать тариф" позволяют 
определить тот или иной способ распределения суммы наряда, опре-
деленной в реквизите "Сумма". При этом каждый из этих влияющих 
факторов можно отключить. Включенная опция "Только прирабо-
ток" приведет к расчету наряда по такой схеме, когда на бригаду 
распределяется только часть суммы, оставшаяся после вычета зара-
ботка по тарифу всех работающих в бригаде.

В шапке документа можно также указать полную производствен-
ную аналитику по отражению стоимости выполненной бригадой 
сдельной работы в составе производственных затрат в оперативном 
учете: 
	- подразделение, в котором производились работы;
	- вид деятельности;
	- продукцию, для производства которой выполнялись опе-
рации;
	- заказ, по которому производилась такая продукция;
	- вид затрат.

Напомним, что обязательным для заполнения реквизитом из вы-
шеперечисленных является только "Вид затрат", остальные реквизи-
ты можно оставить не заполненными. 

Заполнить многострочную часть документа работниками выбран-
ного подразделения (бригады) можно по кнопке "Заполнить". 

Если подразделение не выбрано, то меню кнопки заполнить пред-
лагает два варианта заполнения документа:
	- только производственным персоналом (опция "Только по 
производственному персоналу"), т.е. теми сотрудниками, 
для которых в качестве счета затрат в справочнике "Со-
трудники" установлен счет 23 "Производство";
	- всеми сотрудниками предприятия (опция "По всем 
сотрудникам").

Далее для каждого сотрудника проставляется его КТУ и, если 
необходимо, часы работы.
 
Рассчитать суммы, причитающиеся каждому работнику можно с 
помощью кнопки "Рассчитать". После чего будет заполнен реквизит 
"Результат".

При проведении документа формируются проводки по списанию 
сумм, начисленных каждому сотруднику на счет 23, движение по 
регистру "Производственные затраты" на итоговую сумму наряда.
