Для отражения ежемесячного начисления амортизации по всем 
материальным и нематериальным активам одного подразделения 
(или всем подразделениям) в бухгалтерском учете и ежеквартально-
го по всем объектам - в налоговом учете в типовой конфигурации 
предусмотрен документ "Начисление амортизации". Вариант выполнения 
документа - начисление амортизации по налоговому учету или по 
бухгалтерскому задается выбором соответствующей опции в его диа-
логовой форме.

В бухгалтерском учете начисление амортизации выполняется по 
всем объектам НА, которые числятся в подразделении, указанном в 
реквизите "Подразделение". Если этот реквизит оставить не запол-
ненным - то амортизация начисляется по объектам всех подразделе-
ний.

В налоговом учете износ начисляется по всем объектам учета на 
счетах ОС, НА, т.е. по всем материальным и нематериальным акти-
вам всех подразделений организации. Поэтому реквизит "Подразде-
ление" в этом случае заполнять не нужно. 
 
Рассмотрим особенности начисления амортизации по бухгалтер-
скому и по налоговому учету.

Начисление амортизации в бухгалтерском учете производится 
одновременно по всем материальным и нематериальным необорот-
ным активам, по методу, указанному для каждого объекта в реквизи-
те "Метод расчета износа" в справочнике "Необоротные активы". 

Не начисляется амортизация только по тем объектам, которые нахо-
дятся на модернизации или на консервации (в реквизите "Состоя-
ние" в справочнике необоротных активов для них указано одно из 
перечисленных значений). В справочнике "Необоротные активы" 
должны также быть заданы все параметры, необходимые для приме-
нения того или иного метода, а также счет затрат и вид затрат, на 
которые будет отнесена сумма начисленного износа в корреспонден-
ции с соответствующим субсчетом счета 13 "Износ необоротных ак-
тивов". Если вид затрат для какого-либо объекта в справочнике не 
задан, то при проведении документа используется значение указан-
ного в шапке документа (в реквизите "Вид затрат").

Если в качестве затратного счета для актива в форме справочника 
указан счет 231 "Производство продукции и услуг", сумма начис-
ленного износа по такому объекта отразится в составе производст-
венных затрат и по оперативному учету (по регистру "Производст-
венные затраты").

Начисление амортизации в налоговом учете производится только 
по тем необоротным активам (материальным и нематериальным), 
для которых в форме элемента справочника "Необоротные активы" 
включен флажок "Начислять износ в НУ".

Начисление износа по таким материальным активам осуществля-
ется путем применения норм амортизации, определенных для каж-
дой группы ОФ к балансовой стоимости группы.

Напомним, что группа ОФ, которой принадлежит такой необо-
ротный актив, также определяется в форме элемента справочника 
"Необоротные активы". Нормы амортизации для различных групп 
основных фондов, понижающий коэффициент определяются из 
справочника "Шкала ставок".

При этом амортизация отдельного объекта основных фондов 
группы 1, учет которой ведется на забалансовом счете ОС1, произво-
дится до достижения балансовой такого объекта ста необлагаемых 
налогом минимумов, а амортизация групп 2 и 3 - до достижения 
балансовой стоимости группы нулевого значения.

Величина износа нематериальных активов по налоговому учету 
определяется по нормам, рассчитываемым исходя из их первона-
чальной стоимости и срока полезного использования актива, и на-
числяются до полного погашения стоимости актива. Напомним, что 
учет первоначальной стоимости нематериальных активов, подлежа-
щих амортизации, ведется в типовой конфигурации на забалансовом 
счете НА1.

Квартальная сумма амортизации по нематериальным активам 
рассчитывается исходя из ее месячной нормы, определенной выше-
изложенным образом.

Суммы начисленной амортизации по необоротным материальным 
активам будут отражены по кредиту субсчетов ОС1, ОС2, ОС3, по 
нематериальным активам - по кредиту счета НА2.

	ВНИМАНИЕ! Поскольку кредитовый оборот вышеперечисленных 
	субсчетов используется при построении, в частности, регламен-
	тированного отчета "Декларация о прибыли", то по кредиту 
	этих субсчетов не должны отражаться никакие другие суммы, 
	кроме как суммы амортизационных начислений. Поэтому спи-
	сание балансовой стоимости ликвидируемого необоротного ак-
	тива, в тех случаях, когда это необходимо, следует осуществлять 
	методом "красного сторно" по дебету соответствующего счета 
	учета его балансовой стоимости

С помощью кнопки "Провести" можно осуществить проведение 
документа, не закрывая при этом его диалоговую форму.

После проведения документа становится доступной кнопка 
"Печать", позволяющая получить печатную форму ведомости начис-
ления амортизации по бухгалтерскому или налоговому учету.
