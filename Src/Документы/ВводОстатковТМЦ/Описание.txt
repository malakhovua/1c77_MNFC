Для ввода начальных остатков запасов предприятия как собст-
венных (купленных), так и давальческих (принятых на переработку), 
предназначен документ "Ввод остатков ТМЦ". С помощью этого 
документа можно также ввести остатки как принятой, так и пере-
данной тары. Тип объектов, по которым будут вводиться начальные 
остатки, задается с помощью флажков группы "Вид остатков".

Документы "Ввод остатков ТМЦ" создаются, хранятся и редак-
тируются в журнале "Документы ввода начальных остатков" 
("Журналы"/"Ввод остатков") вместе с другими документами ввода 
начальных остатков. Новый документ вызывается также из меню 
"Документы"/"Ввод остатков"/"Ввод остатков ТМЦ".
 
В шапке документа должны быть заданы место хранение, контр-
агент (обязателен для задания только при вводе остатков давальче-
ских запасов и всех видов принятой тары), а также категория цены 
для установки по умолчанию для каждого объекта. При вводе остат-
ков переданной тары контрагент, которому она была передана, ука-
зывается в реквизите "Место хранения", т.е. реквизит "Контрагент" 
в этом случае можно оставить не заполненным. При вводе остатков 
принятой или переданной тары в реквизите "Вид тары" дополни-
тельно определяется ее вид, а именно, есть ли это: залоговая, воз-
вратная или оборотная тара. В зависимости от вида тары при прове-
дении документа выбирается счет ее учета.

В табличной части документа для всех типов объектов вводятся 
следующие сведения:
	- наименование ТМЦ, остатки по которому вводятся;
	- счет учета ТМЦ (только для собственных запасов);
	- вводимое количество ТМЦ;
	- единица измерения ТМЦ;
	- коэффициент пересчета в базовую единицу
	- учетная цена ТМЦ в гривнях без учета и с учетом НДС. 
	  Для давальческих запасов под учетной ценой понимается 
	  их полная себестоимость;
	- учетная стоимость указанного количества ТМЦ в гривнях 
	  без НДС;
	- учетная стоимость указанного количества ТМЦ в гривнях 
	  с НДС;
	  
При вводе остатков собственных запасов на оптовом складе
счет учета автоматически подставляется из соответствующего
реквизита справочника ТМЦ, но изменение его допускается
(может быть полезным, например, для ввода МПБ в эксплуатации
по счету МЦ).

При вводе остатков собственных запасов на розничном складе 
для каждого ТМЦ дополнительно задается его розничная цена, по 
которой при проведении документа будет рассчитана сумма торговой 
наценки и сформирована соответствующая проводка.

Для давальческих запасов в графах "Цена баланс.", "Сумма ба-
ланс." задаются собственная часть себестоимости единицы и всего 
количества давальческих ТМЦ. При проведении документа на эту 
сумму, формируется проводка по вводу остатков на счет 232 "Услу-
ги по переработке давальческого сырья" (см. подробнее раздел 
"Производство продукции из давальческого сырья"). Полная себе-
стоимость давальческого ТМЦ отражается по соответствующему за-
балансовому счету его учета.

При вводе остатков переданной тары в табличной части докумен-
та задается ее залоговая стоимость. 

При проведении документ формируется соответствующие бухгал-
терские проводки в корреспонденции со счетом 00 и выполняет 
движения по регистрам "Обороты", "Остатки" и "Партии".
