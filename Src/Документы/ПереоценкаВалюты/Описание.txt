Для автоматического определения курсовых разниц по счетам учета 
имущества и обязательств организации, стоимость которых выражена в ино-
странной валюте, в типовой конфигурации предусмотрен документ "Пере-
оценка валюты".

Обращаем внимание на то, что этот документ производит пересчет стои-
мости сразу по всем счетам (в разрезе конкретных объектов 
аналитического учета), у которых установлен признак валютного учета.

При проведении документ проверяет остатки на соответствующих ва-
лютных счетах и сравнивает валютное сальдо и гривневое покрытие по ка-
ждой из валют.

В бухгалтерском учете определенные таким образом курсовые разницы 
будут включены в состав доходов/расходов предприятия. При этом 
курсовая разница по всем счетам и субконто, указанным в табличной 
части документа, будет считаться неоперационной, курсовая разница по 
всем остальным счетам и субконто - операционной.

Например, если в строке табличной части документа указать только 
счет, оставив остальные графы незаполненными, то курсовые разницы, 
определенные  по всем субконто этого счета будут считаться 
неоперацинной, или, если задано первое субконто счета, но не задано 
второе субконто, то курсовые разницы все курсовые разницы, 
определенные по всем остаткам первого субконто в разрезе второго будут 
считаться неоперационными.         

В оперативном учете документ выполнит переоценку остатков по
взаиморасчетам с контрагентами (исключая долги по авансам) и отнесет 
выявленные курсовые разницы в состав доходов/расходов предприятия. 
Полученный доход или убыток можно будет увидеть в отчете "Обороты", 
сформировав его по ТМЦ, указанному в константе "Курсовая разница".

В налоговом учете документом будут сформированы проводки по вклю-
чению в состав валовых расходов (доходов) курсовой разницы, определен-
ной по остаткам на забалансовом счете ВЛ, проводки по валовым дохо-
дам/расходам по балансовой стоимости валюты. Курсовые разницы по ос-
таткам на счетах учета обязательств организации будут включены в состав 
валовых доходов/расходов по задолженностям в валюте. Кроме того, доку-
ментом также будет сформирована соответствующая проводка по переоцен-
ке балансовой стоимости валюты на счете ВЛ.

Признаком формирования проводок по валовым расходам/доходам в 
разрезе указанной аналитики (реквизиты групп "Валовые доходы", 
"Валовые расходы") служит флажок "Делать проводки по валовым 
доходам/расходам".
