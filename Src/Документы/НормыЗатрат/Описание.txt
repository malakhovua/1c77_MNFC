Для задания норм затрат для каждого вида продукции (или зака-
за на ее производство) предназначен документ "Нормы затрат".

Документы "Нормы затрат" создаются, хранятся и редактируются 
в одноименном журнале, который открывается выбором соответст-
вующего пункта меню "Журналы" / "Производство". Новый доку-
мент можно вызвать также с помощью меню "Документы"/ 
/"Производство"/"Нормы затрат" или по кнопке "+" на закладке 
"Продукция" формы элемента справочника "Номенклатура".

В форме документа в первую очередь следует определить наиме-
нование и количество единиц продукции, для которых вы будете 
задавать нормы затрат. 

Если настоящим документом задаются нормы для конкретного 
заказа (см. выше), то в реквизите "Заказ" должна быть установлена 
ссылка на такой документ. Для этого по кнопке   нужно открыть 
журнал заказов и, позиционировав в нем курсор на требуемом доку-
менте, двойным щелчком мыши перенести его в форму документа.
 
Если же задаются нормы затрат, используемые по умолчанию для 
выбранного вида продукции, то реквизит "Заказ" заполнять не нуж-
но. В этом случае при проведении документа информация из его 
табличной части вносится в справочник "Номенклатура" (на заклад-
ку "Продукция"). Там же устанавливается ссылка на текущий доку-
мент, открыть форму которого в дальнейшем можно будет по кнопке 
"...".

В табличной части документа можно указать как детализацию 
статей калькуляции: "Сырье и материалы", "Полуфабрикаты", "Воз-
вратные отходы", так и отдельные статьи калькуляции. 

В первом случае в графе "Вид элемента" из предлагаемого списка 
нужно выбрать: "Материал", "Полуфабрикат" или "Возвратный от-
ход", после чего в графе "Элемент" выбрать из справочника "Но-
менклатура" соответствующий объект и указать его требуемое коли-
чество. Сумма по текущей строке рассчитывается автоматически на 
основании учетной цены матераила, указанной для него в справоч-
нике "Номенклатура", количества и коэффициента пересчета. 

Нормы для статьи калькуляции (в реквизите "Вид элемента за-
трат" выбрано значение "Статья калькуляции") можно задать либо в 
виде суммы, либо в виде процента по отношению к некоторой базо-
вой статье, уже введенной в табличной части документа. Например, 
после ввода суммы основной зарплаты для статьи калькуляции "Ос-
новная зарплата", для статьи "Отчисления на соцстрахование" дос-
таточно указать соответствующий процент (в реквизите "Процент"), 
выбрав предварительно в реквизите "Базовая затрата" значение 
"Основная зарплата". Сумма для такой статьи будет рассчитана ав-
томатически и отобразится в графе "Сумма". Отметим, что реквизи-
ты "Базовая затрата" и "%" доступны только, если в качестве "Вида 
элемента" выбрано значение "Статья калькуляции".

При проведении документ регистрирует себя либо в одноименном 
реквизите документа "Заказ", если он оформлен по конкретному 
заказу, либо в форме соответствующего элемента справочника "Но-
менклатура", как описано выше.

Кнопка "Печать", с помощью которой можно получить печатную 
форму "Нормативного состава продукции", становится доступной 
только после проведения документа.
