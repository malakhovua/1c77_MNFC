Для отражения в учете операций по списанию запасов, недостачи 
которых были выявлены в ходе проведенной инвентаризации, в 
Комплексной конфигурации предназначен документ "Списание 
ТМЦ".

Для ввода нового документа необходимо в меню "Документы" 
главного меню программы выбрать пункт "Складские / "Списание 
ТМЦ".

Форму документа следует начать с выбора склада, с которого бу-
дет произведено списание запасов, вида НДС. Здесь же можно ука-
зать причину списания ТМЦ, необходимую для формирования пе-
чатной формы акта списания.
 
На закладке "Дополнительно" указывается затратный счет, на ко-
торый будет списана учетная стоимость объектов, а также объектов 
аналитики этого счета. Если у организации нет права на включение в 
состав налогового кредита сумм НДС, уплаченных в связи с приоб-
ретением списываемых объектов, нужно включить флажок "На соб-
ственные нужды (нет права на налоговый кредит)". В этом случае 
при проведении документа будет сформирована проводка по дооце-
ниванию себестоимости списываемых ТМЦ и сторнированию нало-
гового кредита.

В табличной части документа указываются собственно наимено-
вания списываемых ТМЦ, с указанием предпочитаемых для списа-
ния в первую очередь партий и количества списываемых единиц. 
Графы "Сумма -", "НДС" и "Сумма +" недоступны для редактиро-
вания, их значения рассчитываются автоматически при проведении 
документа, в соответствии со списываемыми партиями ТМЦ.

С помощью кнопки "Провести" можно осуществить проведение 
документа, не закрывая при этом его диалоговую форму.

Кнопка "Печать", с помощью которой возможно получение 
печатной формы акта списания ТМЦ, становится доступной только 
после проведения документа.

При вводе документа на основании документа "Инвентаризация 
ТМЦ", его форма будет заполнена автоматически теми ТМЦ, кото-
рые необходимо списать по результатам проведенной инвентариза-
ции.

При проведении документ формирует бухгалтерские проводки и 
движения по регистрам "Остатки товаров", "Партии товаров".
