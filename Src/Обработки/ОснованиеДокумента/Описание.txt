В дополнительном окне "Задание основания документа" 
могут содержаться, в зависимости от вида документа, 
следующие управляющие элементы:

Документ-основание - нажав кнопку "…", можно выбрать 
документ, на основании которого будет оформлен 
данный документ. При этом для выбора 
предоставляются только документы с тем же 
контрагентом, который указан в данном документе. 
Если документ-основание уже выбран, нажатие 
кнопки "…" вызывает форму документа-основания. 
Кнопка очистки "Х" предназначена для очистки 
реквизита "Документ-основание". У каждого 
документа может быть не более одного документа-
основания. 

Основание - текстовое поле, в котором записывается 
информация об основании документа. При желании 
пользователь может написать в этом поле любой 
текст. Если выбран документ-основание, то 
информация о документе-основании автоматически 
записывается в это поле. Его содержание будет 
отображаться в печатной форме документа. 

Договор - по кнопке "…" можно выбрать договор, к которому 
будет относиться документ. Выбор договора 
возможен только после выбора контрагента в 
документе и только из договоров этого контрагента. 
Поэтому, если контрагент в документе еще не 
выбран, сначала будет предоставлен на выбор 
справочник контрагентов (причем выбранный 
контрагент становится контрагентом документа). 
Кнопка очистки "Х" предназначена для очистки 
реквизита "Договор". 

Режим оплаты - задается только в платежных документах 
(приходный и расходный кассовые ордера, 
банковская выписка, ввод остатков кредита), 
определяет способ распределения оплаты по 
договорам, может принимать одно из четырех 
значений: "Оплата по конкретному договору (с 
отнесением аванса на договор)", "Оплата по 
конкретному договору (с отнесением аванса без 
указания договора)", "Автораспределение оплаты по 
договорам", "Автораспределение оплаты по договорам 
с указанием приоритетного договора" (подробнее см. 
в документации "Работа по договорам"). 

Оплачивается только по договору - флаг, проставляющийся 
только для некоторых документов (приходная 
накладная, расходная накладная, приходная 
реализатора, расходная реализатора, ввод остатков 
реализации, ввод остатков кредита). Запрещает 
оплату документа: в случае предоплаты - авансом, 
полученным (выданным) без указания договора; в 
случае постоплаты - в режиме автоматического 
распределения оплаты по договорам (подробнее см. 
там же).
