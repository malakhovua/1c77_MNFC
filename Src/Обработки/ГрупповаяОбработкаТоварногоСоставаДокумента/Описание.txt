Окно "Групповая обработка товарного состава документа" 
содержит элементы комплексного изменения (переоценки) 
товарного состава документа. 
Переоценка производится по всем товарам, которые помечены 
в списке товарной номенклатуры обработки (фильтр по товарам). 

Действия с товарным составом документа можно разделить на две 
группы: изменение количества и изменение цены. 

Параметры переоценки задаются выбором действия по 
переоценке товарного состава из списка "Действие" и, при 
необходимости, вводом соответствующего значения с помощью 
кнопки "Значение".

Кроме того, имеются управляющие кнопки:

"Выполнить" - выполняется переоценка товарного состава по 
заданным параметрам без перехода в документ. Таким образом, 
можно произвести, не выходя в форму документа, несколько 
действий по изменению товарного состава. В процессе переоценки 
производимые действия отмечаются в табло сообщений. После 
выполнения каждой переоценки данные в таблице в окне переоценки 
отслеживают произведенные изменения в товарном составе 
документа. 

"Закрыть" - закрытие окна переоценки и возврат в документ. 

Если были произведены какие-либо действия с товарным составом 
по кнопке "Выполнить", то появляется запрос о принятии изменений 
товарного состава документа. 


ФИЛЬТР ДЛЯ ТОВАРОВ

Изменение (переоценка) товарного состава производится по всем 
помеченным товарам. При первом входе в окно переоценки 
помечаются все товары из спецификации документа. 
Отменить или поставить пометку по конкретному товару можно с 
помощью клавиши Enter или двойным щелчком левой кнопки 
мыши. 

Можно снять пометку у всех помеченных ранее товаров с 
помощью кнопки "Снять". С помощью кнопки "Все" можно пометить 
все товары в списке. С помощью кнопки "Инвертировать" можно 
установить ее тем товарам, у которых ее нет, и снять у тех товаров, 
где она стоит. Кнопка "Товары из группы" применяется в том случае, 
если в спецификации документа нужно пометить только товары 
определенной группы (подгруппы). Кнопка "Товары из категории" 
применяется в том случае, если необходимо переоценить только 
товары определенной категории.


ИЗМЕНЕНИЕ КОЛИЧЕСТВА

Возможны следующие варианты изменения:

Удалить: При активизации этого элемента автоматически 
удаляются помеченные позиции состава документа.

Изменить кол-во на (%): количество товара, выписанного в 
документе, по отмеченным позициям изменяется на задаваемое 
количество процентов. Количество процентов задается в окне 
"Значение". Если изменение задается со знаком плюс, то количество 
увеличивается на заданную величину, если со знаком минус, то 
уменьшается.

Изменить кол-во на (в ед. изм.): количество товара, выписанного 
в документе, по отмеченным позициям изменяется на задаваемое 
количество единиц измерения. Количество единиц измерения 
задается в окне "Значение". Если изменение задается со знаком 
плюс, то количество увеличивается на заданную величину, если со 
знаком минус, то уменьшается.

Округлить кол-во до: предоставляется возможность уменьшить 
или увеличить точность задания количества товара. Желаемую 
точность от 0.0001 до 10000 можно выбрать в окне Ввод порядка 
округления количества, вызываемого при нажатии кнопки 
"Значение". 

Добавить из документа. Этот элемент есть во всех документах, 
имеющих товарный состав, в котором можно задавать количество 
товара. Он дает возможность копировать в данный документ 
товарный состав из документа другого вида (например, полезно 
скопировать состав накладной на поступление - в накладную на 
отпуск); этот элемент полезен также при необходимости выписать 
один документ на основании нескольких других (скажем, произвести 
отгрузку по нескольким счетам).

Для копирование нужно выбрать действие "Добавить из 
документа". Станет доступной кнопка "Значение". Ее нажатие 
открывает диалог выбора вида документа. После выбора вида 
документа открывается на выбор журнал документов этого вида.

Копирование состава происходит по следующему алгоритму: для 
каждого товара в составе выбранного документа проверяется, 
имеется ли уже такой же товар с такой же упаковкой в составе 
документа. Если есть, то просто увеличивается количество в 
соответствующей товарной позиции. Если нет, то в товарном составе 
документа добавляется новая строка, в которой количество и 
упаковка берется из выбранного документа. Цены для добавляемых 
позиций не копируются, а устанавливаются в соответствии с типом 
цен модифицируемого документа. Если тип цен в документе не 
задан, или у добавляемого товара не задана цена нужного типа, то 
ценовые характеристики новых позиций не заполняются. 

ВНИМАНИЕ!!! При копировании состава не производится проверка, 
получаются ли у строк с одинаковыми товарами одинаковые 
цены (если учесть коэффициенты пересчета единиц измерения 
по позициям) с целью обеспечения возможности проведения 
документа. Предполагается, что ответственность за это 
совпадение берет на себя пользователь, предпринимающий 
копирование состава. 

Элемент "Добавить из документа" работает независимо от 
установленного фильтра (помеченных позиций).


ИЗМЕНЕНИЕ ЦЕНЫ

Вслед за нижеописанным изменением цены товаров в составе 
документа в процессе переоценки происходит автоматический 
пересчет суммы по товарной позиции и суммы НДС.

Возможны следующие варианты изменения цен товаров в составе 
документа:

Изменить цену на (в %): цена товара изменяется на 
определенный процент. Величина процента задается в специальном 
окне, который вызывается с помощью кнопки "Значение". Если 
процент задан с плюсом, то происходит увеличение цены, а если с 
минусом, то ее уменьшение. Изменение цены происходит только по 
помеченным товарам.

Изменить цену на (в ед. валюты): цена товара изменяется на 
задаваемую сумму в валюте документа. Величина суммы задается в 
специальном окне, которое вызывается с помощью кнопки 
"Значение". Если сумма задана с плюсом, то происходит увеличение 
цены, а если с минусом, то ее уменьшение. Изменение цены 
происходит только по помеченным товарам. 

Разнести сумму по кол-ву - разделить заданную сумму на 
несколько частей, так, чтобы величина каждой части соотносилась с 
величиной остальных следующим образом: суммы по позициям 
изменяются пропорционально количеству товаров (при этом цены 
товаров изменяются на одинаковую величину для единицы 
измерения каждого товара, независимо от того, что это за единица). 
Произвольная сумма в валюте документа вводится в специальном 
окне, который вызывается при нажатии кнопки "Значение". 

Разнести сумму по суммам - разделить заданную сумму на 
несколько частей, так, чтобы величина каждой части соотносилась с 
величиной остальных следующим образом: суммы по позициям 
спецификации изменяются пропорционально самим суммам (при 
этом цены по позициям изменяются пропорционально самим ценам). 
Произвольная сумма в валюте документа вводится в специальном 
окне, который вызывается при нажатии кнопки "Значение".

Округлить цены до - предоставляется возможность уменьшить 
или увеличить точность задания цен на товары. Желаемую точность 
от 0.0001 до 10000 можно выбрать в окне "Ввод порядка округления 
цены", вызываемого при нажатии кнопки "Значение". 

Заполнить цены из справочника - при нажатии на кнопку 
"Значение" предоставляется выбор типа цены, с тем, чтобы при 
выполнении переоценки цены всех помеченных товаров в товарном 
составе были заменены ценами, указанными в данном типе цены для 
конкретной упаковки конкретного товара. При этом они 
пересчитываются из валюты, заданной в типе цены, в валюту 
документа. Для пересчета используется курс документа для валюты 
документа и курс в справочнике валют на дату курса документа для 
валюты цены - внимание! именно на дату курса, а не на дату 
документа. Производится также пересчет из упаковки, заданной в 
типе цены, в упаковку в документе (по коэффициентам пересчета 
количества для этих упаковок данного товара).
