Программа позволяет корректировать и обрабатывать 
одновременно группу элементов справочника контрагентов. 

После задания фильтров и нажатия кнопки "Выбрать" на экране 
появляется окно со списком элементов справочника, отобранных для 
групповой обработки. 

Список не иерархический: в нем могут быть представлены 
элементы из всех групп справочника. Группа, к которой относится 
элемент, указана в одной из колонок списка.

В списке можно пометить какие-то из элементов с помощью 
клавиши Enter или двойного щелчка мыши, и провести необходимые 
действия одновременно со всеми помеченными элементами. Чтобы 
пометить объект, необходимо установить курсор на любое поле в 
списке, кроме поля "Пометка".

В окне списка можно также пользоваться кнопками:

"Все" - помечаются все элементы в списке,

"Снять" - снимается пометка со всех отмеченных элементов,

"Инвертировать" - снимается пометка у всех элементов, у 
которых она есть, и ставится тем, у которых ее нет.

В выпадающем списке "Действие" можно выбрать действия, 
которые будут выполнены по отношению к помеченным элементам 
справочника (действия различаются для разных справочников). Для 
некоторых действий можно (или обязательно нужно) установить 
дополнительные параметры с помощью кнопки "Значение".

Для групповой обработки контрагентов можно выбрать одно из 
следующих действий, которое будет произведено над всеми 
помеченными элементами списка:

Пометить на удаление - позиции будут помечены на удаление.

Снять пометку удаления - у позиций, помеченных на удаление, 
будет снята эта пометка.

Перенести в группу - в поле "Значение" необходимо выбрать 
соответствующую группу справочника контрагентов. 
Позиции будут перенесены в указанную группу. 

Установить валюту взаиморасчетов - в поле "Значение" 
необходимо выбрать из справочника валют валюту, 
которая будет установлена как валюта 
взаиморасчетов.

Установить тип цен покупателю - в поле "Значение" выбрать 
тип цен справочника типы цен. У всех контрагентов 
установится тип цен покупателю по умолчанию, 
равный заданному. 

Установить тип цен поставщика - в поле "Значение" выбрать 
тип цен справочника типы цен. У всех контрагентов 
установится тип цен поставщика по умолчанию, 
равный заданному. 

Установить вал. кред. пост. с пересчетом (валюту кредита 
поставщику с пересчетом суммы) - в поле 
"Значение" необходимо выбрать из справочника 
валют валюту кредита поставщика. У всех 
контрагентов будет установлена эта валюта кредита 
поставщика, при этом сумма кредита поставщика 
будет пересчитана из старой валюты в новую по 
курсу на текущий момент. 

ВНИМАНИЕ!!! После проведения данной операции необходимо 
перепровести все документы.

Установить вал. кред. пост. без пересчета (валюту кредита 
поставщику без пересчета суммы) - в поле 
"Значение" необходимо выбрать из справочника 
валют валюту кредита поставщика, которая и будет 
установлена у всех контрагентов, при этом сумма 
кредита поставщика не изменится.

ВНИМАНИЕ!!! После проведения данной операции необходимо 
перепровести все документы.

Установить сумму кред. пост. (кредита поставщику) - в поле 
"Значение" необходимо ввести число, равное сумме 
кредита поставщику. У всех контрагентов будет 
установлена сумма кредита поставщику, равная 
установленному значению, независимо от того, какая 
валюта кредита поставщику у контрагента. 
Установить глубину кред. пост. (кредита поставщику) - в поле 
"Значение" необходимо ввести число, равное глубине 
кредита поставщику. У всех контрагентов будет 
установлена глубина кредита поставщику, равная 
установленному значению. 

Аналогичные действия проводятся по валюте расчета с 
покупателем, глубине и сумме кредита покупателю.

Установить категорию - в поле "Значение" выбрать категорию 
из справочника категорий. Если такой категории не 
было в списке категорий по данной позиции, то она 
добавится.

Удалить из категории - в поле "Значение" выбрать категорию 
из справочника категорий. Если такая категория была 
у данной позиции, то она удалится из списка 
категорий по позиции.

Удалить из всех категорий - в этом случае удалятся все 
категории из списка категорий по всем позициям.

Установить комментарий - установить примечание во всех 
отобранных позициях. Текст вводится в окне, 
появляющееся при нажатии кнопки "Значение". 

По кнопкам:

"Выполнить" - выполняется обработка помеченных элементов 
справочника,

"Закрыть" - выход без обработки в окно настройки.
