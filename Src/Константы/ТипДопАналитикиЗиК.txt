Тип дополнительной аналитики в конфигурации "Зарплата + Кадры для Украины"
на счете 231 "Производство продукции и услуг". 

Может принимать следующие значения:
	0 - без доп. аналитики
	1 - доп. аналитикой будут заказы
	2 - доп. аналитикой будет продукция
    
Эта константа используется только в случае, если в константе "Рассчитывать 
зарплату в конфигурации" установлено значение 0 (т.е. зарплата расситывается
в конфигурации "Зарплата + Кадры для Украины").
