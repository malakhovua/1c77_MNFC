Хранит суммовые остатки производственных затрат 
(общей производственной себестоимости продукции). 

Остаток на конец месяца - незавершенное производство. 

Движения по приходу - отражение фактически произведенных 
затрат по выпуску продукции, а также перераспределение затрат 
в конце месяца. Движения по расходу - отражение фактической 
себестоимости выпущенной продукции (в конце месяца).

Измерения
	Фирма - Здесь указывается значение фирмы..
	ВидДеятельности - Вид деятельности. Может быть не выбран
	Подразделение - Подразделение. Может быть не выбрано
	Продукция - Продукция. Может быть не выбрана
	ВидЗатрат - Вид затрат.
	Заказ - Документ "Заказ", по которому производится продукция.
Может быть не выбран
	Материал - Материал, который был списан на продукцию 
в качестве прямых затрат. Может быть не выбран

Ресурсы
	Количество - Количество материала. Указывается только если 
заполнено измерение Материал
	Сумма - Сумма затраты.
	СуммаДав - Сумма давальческой составляющей затраты (в том числе).

Реквизиты
	КодОперации - Код производственной операции.
