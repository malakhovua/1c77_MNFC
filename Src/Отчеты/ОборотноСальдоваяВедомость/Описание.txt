Оборотно-сальдовая ведомость содержит для каждого счета ин-
формацию об остатках на начало и на конец периода и оборотах по 
дебету и кредиту за установленный период. Этот отчет (в просторе-
чии - "оборотка") является одним из наиболее часто используемых 
отчетов в бухгалтерской практике.

Для вывода данной ведомости выберите пункт "Оборот-
но-сальдовая ведомость" меню "Отчеты" главного меню программы.

Вывести автоматически обновляемый аналог оборотно-сальдовой 
ведомости по интересующим счетам можно также используя табло 
счетов (пункт "Табло счетов" меню "Сервис" главного меню про-
граммы).

При выборе пункта "Оборотно-сальдовая ведомость" на экран будет 
выдана диалоговая форма запроса параметров оборотно-сальдовой 
ведомости, в которой следует указать:
	- за какой период необходимо сформировать оборот-
но-сальдовую ведомость. Период задается выбором даты 
начала периода и даты окончания периода;
	- составлять ли ведомость в целом по счетам (флажок 
"Данные по субсчетам" снят) и дополнительно показы-
вать остатки и обороты на субсчетах (флажок "Данные по 
субсчетам" установлен);
	- показывать ли для счетов с признаком валютного учета 
остатки и обороты только в гривнях (флажок "Данные по 
валютам" снят) или дополнительно по каждой валюте в 
отдельности (флажок "Данные по валютам" отмечен). В 
первом случае в строках ведомости будет отражаться 
суммарный гривневый эквивалент валютных сумм, во 
втором- дополнительно отображаются данные в валюте;
	- отображать ли развернутое сальдо, согласно "правилам", 
устанавливаемым на закладке "Развернутое сальдо" диа-
логовой формы запроса параметров отчета.

В спецификации табличной части закладки "Развернутое сальдо" 
содержится список правил, по которым будет построено развернутое 
сальдо. Для каждого правила должны быть заданы следующие пара-
метры: 
	- счет, по которому будет сформировано развернутое саль-
до;
	- вариант формирования развернутого сальдо: по субсчетам 
(для этого следует дважды щелкнуть мышью в поле "Сс", 
после чего там появится символ "Х") или по уровням 
аналитики. Для определения тех видов субконто, в разре-
зе которых должно быть сформировано развернутое саль-
до, необходимо произвести двойной щелчок мышью в со-
ответствующих полях ("Ан.1", "Ан.2", "Ан.3"). Отметим, 
что при выборе счета, по которому, к примеру, ведется 
аналитический учет только в разрезе одного вида субкон-
то в графах Ан2 и Ан3 будут стоять символы "-", и их ре-
дактирование будет недоступно.

Для внесения нового правила следует воспользоваться кнопкой 
"+" . Кнопка "=" служит для редактирования уже введенного правила, 
кнопка "Х" - для удаления правила, на котором в текущий момент 
позиционирован курсор.

После заполнения всех необходимых параметров настройки, на-
жмите на кнопку "Сформировать" или на кнопку "ОК". Программа 
сформирует оборотно-сальдовую ведомость и выведет ее для про-
смотра.
