Для получения информации об оборотах компании по закупкам и 
продажам, анализа себестоимости закупаемых ТМЦ (запасов) пред-
назначен отчет "Обороты". 

С помощью параметров настройки отчета можно произвести раз-
личный отбор и группировку параметров отчета. 

В отчет можно отобрать информацию в самых различных разре-
зах: по конкретным ТМЦ, группам ТМЦ, ТМЦ с определенным сче-
том учета, категориям ТМЦ, а также по произвольному списку всех 
перечисленных признаков (множественный фильтр). 

Отчет может быть построен по конкретному поставщику, группе 
поставщиков, по поставщикам, имеющим определенную категорию 
(аналитический признак), поставщикам определенного региона. С 
помощью множественного фильтра можно построить отчет по спи-
ску произвольно выбранных поставщиков или отобрать тех постав-
щиков, которые имеют вполне определенный список различных ка-
тегорий (аналитических признаков).

Точно таким же образом можно произвести отбор и по покупателю.

Предусмотрена также возможность детализации отчета по раз-
личным параметрам: по позициям номенклатуры, по покупателям, по 
поставщикам, счету учета, регионам поставщика и покупателя, видам 
деятельности, партиям и документам движений. Кроме того, можно изме-
нять  порядок представления данных в отчете. Порядок представления 
данных изменяется в специальном окне с помощью управляющих 
кнопок (вверх, вниз). Установив курсор на нужную позицию, можно 
с помощью управляющих кнопок переместить ее в нужное место.

Изменяя степень детализации и порядок представления данных 
можно получать самую разнообразную информацию в отчете по про-
дажам. Можно, например, получить информацию о том, каким поку-
пателям были проданы товары поставщиков. Для этого нужно уста-
новить детализацию по номенклатуре, поставщикам и покупателям, 
порядок представления данных: "номенклатура, поставщик, покупа-
тель". Порядок представления данных "поставщик, покупатель, но-
менклатура" даст возможность получить данные о том, как продава-
лись товары конкретных поставщиков (каким покупателям какие 
товары и сколько).

С помощью флажков, расположенных в нижней части закладки 
"Основные", можно определить включать ли в отчет информацию об 
общей стоимости и количестве прихода товара (флажок "закупка"), 
продажи товара (флажок "продажа"), полученного дохода (флажок 
"доход").

Отметим, что порядок и состав данных в отчете зависит от ука-
занной детализации и порядка представления данных. Если детали-
зация не указана, то выводятся итоговые данные. 

В итоговых данных выводится следующая информация:
	Приход - общее количество и общая сумма по всем документам 
прихода в соответствии с установленным в отчете от-
бором по номенклатурным единицам, контрагентам и пр.
	Расход -  общее количество и общая сумма по себестоимости 
всех проданных ТМЦ, в соответствии с установлен-
ным в отчете фильтрам.
	Доход - общая прибыль. Прибыль считается как разница между 
суммой полученной при реализации ТМЦ в ценах 
продажи и суммой, затраченной на приобретение то-
вара, т.е. сумма реализованных ТМЦ по себестоимо-
сти. 

При полной детализации выводятся следующие данные: полное 
наименование и количество проданного товара, наименование поку-
пателя, кому был продан товар, и наименование документа, по кото-
рому производилась продажа, наименование поставщика, товар ко-
торого был продан, партия товара и документ движения. Порядок
представления данных настраивается.

Отметим, что производительность отчета при выборе группировки
по партиям резко снижается. В этом случае рекомендуется по воз-
можности ограничивать выбираемые данные, устанавливая значения
фильтров.
